Subject: APPT LIST REPORT - SCHEDULING REPLACEMENT

Category: 
  - Routine

Description:
============

  
 The Appointment List [SDLIST] report has been modified to:
  
 1. Include the patient's phone number on the report.  The phone number
 is field #.131 in the PATIENT (#2) file: PHONE NUMBER [RESIDENCE].
  
 2. Include the service connected percentage on the report for those 
 patients who are service connected.  The service connected percentage is
 field #.302 in the PATIENT (#2) file: SERVICE CONNECTED PERCENTAGE.
  
 3. Retrieve appointment data by calling Scheduling Replacement's 
 appointment query: SDAPI^SDAMA301.

 INSTALLATION INSTRUCTIONS
 =========================
 This patch should be installed during non-peak hours. Users can be on
 the system. Installation will take less than 5 minutes.
  
 NOTE: SD*5.3*80, SD*5.3*301, and SD*5.3*373 are REQUIRED
 PATCHES. KIDS will not allow the installation of this patch without prior
 installation of these patches.
  
  
   1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu.
  
  
   2.  From the Kernel Installation and Distribution System Menu, select
       the Installation menu.
  
   3.  From this menu, you may elect to use the following options
       (when prompted for INSTALL NAME, enter SD*5.3*266):
         a.  Backup a Transport Global - this option will create a backup
             message of any routines exported with the patch.  It will NOT
             backup any other changes such as DDs or templates.
         b.  Compare Transport Global to Current System  - this option will
             allow you to view all changes that will be made when the patch
             is installed.  It compares all components of the patch
             (routines, DDs, templates, etc.).
         c.  Verify Checksums in Transport Global - this option will
             allow you to ensure the integrity of the routines that are in
             the transport global.
  
   4.  Use the Install Package(s) option and select the package SD*5.3*266.
  
   5.  When prompted 'Want KIDS to INHIBIT LOGONs during the install?
       YES??//',respond NO.
  
   6.  When prompted 'Want to DISABLE Scheduled Options, Menu Options, and
       Protocols? YES//', respond YES and disable the following option:
  
       Appointment List [SDLIST]
