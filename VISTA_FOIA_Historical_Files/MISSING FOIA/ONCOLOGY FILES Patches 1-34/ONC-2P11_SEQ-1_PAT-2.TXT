Subject: Fix UNDEF in FOLLOWUP

Category: 
  - Routine

Description: 
============

 This patch fixes a problem reported in NOIS number BRK-0396-10177.
 When the Tumor Registrar generated a follow-up letter, if the CONTACT
 was deleted, it resulted in an UNDEFINED VARIABLE error.  This patch
 will fix the undefined variable error.
 
 
 INSTALLATION INSTRUCTIONS: This patch should be installed when the ONCOLOGY
 users are off the system.  There are no patch dependencies associated with
 this patch.  If this routine is mapped at your site, remember to disable
 mapping before installing the patch and to re-enable it when you are finished.
 The routine included in this patch should be installed in the production UCI.
 To install the patch, unpack the PackMan patch message into your production
 account.  The estimated installation time for this patch is one minute.  The
 ONCOLOGY options do not need to be disabled during the installation of this
 patch.  Your customer support representative will answer any questions
 regarding this patch.

Routine Information:
====================

Routine Name:
  - ONCOFUP

 The second line should now look like the following:
 ;;2.11;ONCOLOGY;**2**;Mar 07, 1995
