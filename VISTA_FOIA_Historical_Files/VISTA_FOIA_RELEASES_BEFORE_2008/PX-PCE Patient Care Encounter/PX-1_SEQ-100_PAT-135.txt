Subject: PCE VIEW APPOINTMENT HEADER FIX

Category: 
  - Print Template

Description:
============

 The PCE View Appointment List not displaying the data range or GAF status.
  
 ASSOCIATED NOIS:
 --------------- 
 MOU-0701-31544  ENCOUNTER VIEWER DROPS DATE RANGE POST PATCH 78

 NOIS OVERVIEW:
 -------------
  
 MOU-0701-31544  ENCOUNTER VIEWER DROPS DATE RANGE POST PATCH 78
 PUG-0204-51068  PCE LIST MANAGER SCREEN DISPLAY
  
 Problem:
 -------
  
 Of the 3 menu options mentioned in PX*1*78, PXCE ENCOUNTER VIEWER 
 does not reveal the date range underneath the PC Provider/Team
 information. The other two options are fine.
  
 Also, the List by Encounter action under the PCE View Appointment
 List and PCE Appointment List header information drops from screen. 
  
     
 Resolution:
 ----------
 This patch modifies the header on the PCE View Appointment List by 
 increasing the header lines by 2 lines to allow for the date range
 and GAF status when there is a Provider for the patient.
  
 This patch also modifies the header on the List by Encounter by
 increasing the header by 2 lines to allow for date range and GAF
 status when there is a Provider for the patient.
