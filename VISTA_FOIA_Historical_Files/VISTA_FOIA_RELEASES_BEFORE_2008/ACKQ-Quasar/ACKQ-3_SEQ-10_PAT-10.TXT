Subject: DECEASED PATIENTS BEING CHECKED OUT

Category:

Description:
============

 This patch will correct the problem in QUASAR
 (Quality Audiology and Speech Pathology Audit and Review)
  that allows the user to create a new visit for deceased patients.

 NOIS/REMEDY OVERVIEW:
 =====================
  
 1)ANN-0604-41062 Deceased Patients Being Checked Out / HD63990
  
 Problem:
 --------
 The QUASAR system currently allows users to create new visits
 for deceased patients when using the option New Clinic Visits
 [ACKQAS VISIT ENTRY].
  
 Resolution:
 -----------
 Modify routine ACKQAS to check to see if the patient is deceased.
 If the patient has passed away then display the message 
 [PATIENT DIED ON mm/dd/yy], clear the variables and return to the Patient
 prompt.
