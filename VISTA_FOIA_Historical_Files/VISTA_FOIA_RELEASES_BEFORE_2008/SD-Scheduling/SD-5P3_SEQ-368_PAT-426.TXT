Subject: ERROR CODES NOT RETURNED ON DOCUMENTED NODE.

Category: 
  - Routine

Description:
============

 The Appointment Filters API routine is documented to return error code 
 values in the ^TMP($J,"SDAMA301","ERROR",error_code) global. 
 This is incorrect.
 The error codes are returned in the ^TMP($J,"SDAMA301",error_code) global.
  
 ASSOCIATED REMEDY TICKET:
 ========================
 HD64821 - $$SDAPI^SDAMA301(.ARY) doesn't behave as documented.
  
 ASSOCIATED E3R(s):
 =================
 N/A
  
 PARTICIPATING TEST SITES:

 Puget Sound HCS
 El Paso, TX
  
 REMEDY OVERVIEW:
 ===============
 HD64821 - $$SDAPI^SDAMA301(.ARY) doesn't behave as documented.
  
 Problem:
 --------
 The routine SDAMA301 is documented to return error code values in the 
 ^TMP($J,"SDAMA301","ERROR",error_code) global. This is not correct. 
 The routine SDAMA300 sets the error code values in
 ^TMP($J,"SDAMA301",error_code) global. 
  
 Resolution:
 ----------
 The documentation in the SDAMA301 routine will be changed to reflect the 
 correct storage.
