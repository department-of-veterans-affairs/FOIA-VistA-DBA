Subject: ERROR IN SNAPSHOT RPT

Category: 
  - Routine

Description: 
============

 Chicago HCS reported the Snapshot of the Program Today [WV PRINT SNAPSHOT]
 output had the same counts when the report was sorted by calendar or
 fiscal year. This bug is caused in the WVRPSNP routine when it sets the
 variable WVJDT to the FileMan date for the start of the fiscal/calendar
 year. This bug only occurs when the report is run in October.
  
 Associated NOIS: ISH-1099-41557
