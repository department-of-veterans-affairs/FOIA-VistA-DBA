Subject: DOL TYPE AND SOURCE CODES UPDATE

Category: 
  - Data Dictionary
  - Routine

Description: 
============

 The sole purpose of this patch is to provide table updates for:
  
 ASISTS DOL TYPE OF INJURY CODES (#2263)
 ASISTS DOL SOURCE OF INJURY CODES (#2263.1)
 ASISTS DOL CAUSE OF INJURY CODES (#2263.2)
  
  
 This patch addresses the following E3R(s):
 ------------------------------------------
 There are no E3Rs associated with this patch
  
   
 This patch addresses the following NOIS message(s):
 ---------------------------------------------------
 There are no NOIS messages associated with this patch
  
  
 ** NOTE **
    ASISTS 2.0 (OOPS GUI VERSION) must be installed prior to installing 
    this patch. 
  
  
 The ASISTS DOL TYPE OF INJURY CODES (#2263) has been updated to include
 the following six new requested codes:
  
    Overstress, Exhaustion, Overexertion - 630
    Vaccination - 740
    Preventative Therapy - 750
    Decontamination Agent - 760
    Terrorism Incident (e.g., declared by FBI) - 770
    Insufficient Oxygen - 780
  
  
 The ASISTS DOL SOURCE OF INJURY CODES (#2263.1) has been updated to include
 eleven new requested codes, and three updates to existing codes.
  
 New Codes:
  
    Decontamination Agent - 725
    Smallpox - 753
    Anthrax - 754
    For Other WMD/terrorism agents or contaminated victims - 755
    For health care associated pathogens - 756
    For diseases relative to community prevalence - 757
    Needle (Used) - 832
    Sharp (e.g. Scalpels and Lancets) Clean - 833
    Sharp (Used) - 834
    Needle/Sharp from a disposal container - 835
    Decontamination equipment (full ensemble) - 1050
  
 Updates:
   
    OLD - Needle     831
    NEW - Needle (Clean)     831
  
    OLD - Human (communicable disease)   950
    NEW - Human (communicable disease, e.g. Smallpox)   950
  
    OLD - Biological agent   960
    NEW - Bacteria, Virus (Not Human Contact, e.g. Anthrax)  960
  
  
 The ASISTS DOL CAUSE OF INJURY CODES (#2263.2) had the following 
 new code added: 
  
    Reaction to Smallpox Vaccination - 1S

   ==================  INSTALLATION INSTRUCTIONS ==================
  
 If installed during the normal workday, it is recommended that the 
 following selections in the OPTION (#19) file, and all of their 
 descendants be disabled to prevent possible conflicts while running the 
 KIDS Install. Other VISTA users will not be affected.
  
 ** It is recommended that this patch be installed when no ASISTS users are
    on the system.
  
 Install time: Less than 5 minutes.
  
  
 1.   LOAD TRANSPORT GLOBAL
      ---------------------
      Choose the PackMan message containing this patch and invoke the
      INSTALL/CHECK MESSAGE PackMan option. 
  
 2.   DISABLE ROUTINE MAPPING   (DSM for Open VMS sites only) 
      -----------------------
      Disable routine mapping on all systems for the routines listed in 
      step 3 below.  
       
      NOTE:  If the routines included in this patch are not currently in 
             your mapped routine set, please skip this step.
  
 3.   COMPONENTS SENT WITH PATCH
      --------------------------
      The following is a list of the routines included in this patch. 
      The second line of each of these routines now looks like: 
  
          <tab>;;2.0;ASISTS;**1**;Jun 03, 2002
  
                       CHECK^XTSUMBLD results
  
      Routine          Before Patch     After Patch    Patch List 
      -------          ------------     -----------    ----------
      OOPSXV21           *NEW*            *DELETED          1
  
 4.   START UP KIDS
      -------------
      Start up the Kernel Installation and Distribution System Menu
      [XPD MAIN]:
  
          Edits and Distribution ...
          Utilities ...
          Installation ..
  
      Select Kernel Installation & Distribution System Option:  INStallation
                                                               ---
          1       Load a Distribution
          2       Verify Checksums in Transport Global
          3       Print Transport Global
          4       Compare Transport Global to Current System
          5       Backup a Transport Global
          6       Install Package(s)
                  Restart Install of Package(s)
                  Unload a Distribution
  
 5.   Select Installation Option:
      ---------------------------
      NOTE:  The following are OPTIONAL, and since there are
             no existing routines with this patch, a is not needed.
             (When prompted for the INSTALL NAME, enter OOPS*2.0*1):
  
      a.  Backup a Transport Global - This option will create a backup
          message of any routines exported with the patch. It will
          not backup any other changes such as DD's or templates.
      b.  Compare Transport Global to Current System - This option
          will allow you to view all changes that will be made when
          this patch is installed. It compares all components of this
          patch (routines, DD's, templates, etc.).
      c.  Verify Checksums in Transport Global - This option will
          allow you to ensure the integrity of the routines that are in
          the transport global.
  
 6.   INSTALL KIDS PATCH
      ------------------
      **This is the step to start the installation of this KIDS patch:
  
      a. Choose the Install Package(s) option to start the patch
         install.
      b. When prompted 'Want KIDS to INHIBIT LOGONs during the
         install? YES//' answer NO.
      c. When prompted 'Want to DISABLE Scheduled Options and Menu
         Options and Protocols? YES// answer NO.
  
 7.   REBUILD MAPPED ROUTINE(S)  (DSM for Open VMS sites only)
      -------------------------
      Optional - Include the routines distributed with this patch
      in the mapped routine set.
      
      NOTE: This step is only necessary if you performed step 2 or 
            if you wish to include the routines in your mapped set.
