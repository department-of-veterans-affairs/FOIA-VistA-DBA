 Patch XU*8*289 updates option 'Grant Access by Profile' [XUSERBLK]
 with the following:
  
  1) Copy non-expiration PERSON CLASS of the template person to the new 
     user.
  2) Copy DELETE ALL MAIL ACCESS of the template person to the new user.
  3) Copy DELETE KEYS AT TERMINATION of the template person to the new
     user.
  4) Set CPRS Tab EFFECTIVE DATE of the new user to be the current date
     for the non-expiration CPRS Tab.
