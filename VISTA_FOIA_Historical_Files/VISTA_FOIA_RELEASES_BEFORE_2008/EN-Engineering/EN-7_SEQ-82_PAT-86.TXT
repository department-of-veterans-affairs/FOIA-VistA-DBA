Subject: ADD AMAFIT TO THE NX FUND (#6914.6) FILE

Category: 
  - Other

Description:
============

 This patch adds fund code AMAFIT to the NX FUND (#6914.6) file.

 REMEDY OVERVIEW:
 ================
 1. 133278     Cannot change entry to AMAFIT fund
    
    Problem:
    --------
    The FY 2006 Military Construction and Veterans Affairs and Related
    Agencies Appropriation Act created a new Information Technology
    appropriation (0167BM) for non-pay IT items.  In order to accommodate
    this, a new fund AMAFIT will need to be added to the NX FUND file
    (#6914.6).
    
    Resolution:
    -----------
    This patch adds Fund AMAFIT (INFORMATION TECHNOLOGY) as an entry in the NX
    FUND file (#6914.6).
  
    
 TECHNICAL:
 ==========
 The following new fund will be added to the NX FUND (#6914.6) file:
    
    NX
    FUND                                                      ASSOCIATED
    CODE     BRIEF DESCRIPTION             ABBREVIATION           FUND
    ----------------------------------------------------------------------
    
    AMAFIT   AMAF Info Tech Fund 0167        INFO TECH               0167
