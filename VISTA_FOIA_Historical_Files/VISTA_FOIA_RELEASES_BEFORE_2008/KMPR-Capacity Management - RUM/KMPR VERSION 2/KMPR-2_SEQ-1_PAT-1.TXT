Subject: Routine Updates

Category: 
  - Routine
  - Enhancement (Mandatory)

Description:
============

  
 1. The Status of RUM Collection [KMPR STATUS COLLECTION] option has been
    updated.  It now uses List Manager for it's display. 
 2. KMPU* routines are removed.  These routines have been replaced by
    similar KMPDU* routines already in use.
 3. ENTRY ACTION code removed from option RUM Manager Menu [KMPR RUM
    MANAGER MENU].
 4. The following menus were designed for MSM and are no longer in use, so
    they are being deleted: 
      [XUCS MANAGER MENU]
      [XUCS MANUAL PURGE OF DATA]
      [XUCSR REPORTS MENU]
      [XUCS SITE EDIT MENU]

 INSTALLATION:
 =============
  
  
 NOTE: This patch should be installed during off hours, when user activity 
 is low. It takes less than five minutes to install this patch, but you may
 queue it if you wish. Users may be on the system during installation of 
 this patch and you do not need to stop TaskMan.
  
 1.  Use the 'INSTALL/CHECK MESSAGE' option on the PackMan menu. This
        option will load the KIDS package onto your system.
  
 2.  On the PackMan menu, use the 'INSTALL/CHECK MESSAGE' option.
     This loads the patch into a Transport Global on your system.  
  
 3.  On the KIDS:Installation menu, use the following options to check and 
 install the Transport Global.
  
        Verify Checksums in Transport Global
        Print Transport Global
        Compare Transport Global to Current System
        Backup a Transport Global
        Install Package(s)
