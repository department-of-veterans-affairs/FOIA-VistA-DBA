Subject: Daily Timing Transmission

Category: 
  - Data Dictionary
  - Routine

Description:
============

 1. Timing data will no longer be transmitted weekly.  The daily 
    transmission will continue.  Weekly stats will be compiled at the CP 
    National Database site (using daily data).
 2. Weekly background data is removed from the status report [KMPD STATUS] 
    option.
 3. Option [KMPD CM DEVELOPER TOOLS] CM Developer Tools is included.  This 
    is a Broker type option, and is intended for use with a future release 
    of the CP Code Evaluator.
 4. File #8972.3 (CP DATA ELEMENTS) is included.  This is intended for use 
    with a future release of the CP Code Evaluator.  *** ENTRIES IN THIS 
    FILE SHOULD NOT BE CHANGED ***
 5. The following is a list of REMOTE PROCEDURES that are included.  These 
    are intended for use with a future release of the CP Code Evaluator:
     KMPD ASSOC ROUTINE ADD/DEL              
     KMPD ENVIRONMENT                        
     KMPD ERROR LOG DATA                     
     KMPD ERROR LOG DATE                     
     KMPD FILE INQUIRY                       
     KMPD FILE SEARCH                        
     KMPD FM DATE                            
     KMPD GLOBAL LIST                        
     KMPD ROUTINE FIND          
     KMPD ROUTINE INQ           
     KMPD ROUTINE LINE SRCH     
     KMPD ROUTINE SAVE          
     KMPD ROUTINE STATS         
     KMPD VERSION               
 6. The following fields have been added to file #8973 (CP PARAMETERS):
     #1.101 SAGG TRANS TO TYPE [1S]
     #1.102 SAGG TRANS TO OPTION [2F]
     #1.103 SAGG TRANS TO DOMAIN [3P:4.2]
     #2.101 RUM TRANS TO TYPE [1S]
     #2.102 RUM TRANS TO OPTION [2F]
     #2.103 RUM TRANS TO DOMAIN [3P:4.2]
     #3.101 HL7 TRANS TO TYPE [1S]
     #3.102 HL7 TRANS TO OPTION [2F]
     #3.103 HL7 TRANS TO DOMAIN [3P:4.2]
     #4.101 TIMING TRANS TO TYPE [1S]
     #4.102 TIMING TRANS TO OPTION [2F]
     #4.103 TIMING TRANS TO DOMAIN [3P:4.2]
     *** THESE FIELDS SHOULD NOT BE CHANGED BY THE FACILITY ***
