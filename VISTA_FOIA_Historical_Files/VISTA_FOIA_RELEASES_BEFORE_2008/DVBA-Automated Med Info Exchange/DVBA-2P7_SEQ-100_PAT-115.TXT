 No new functionality is being provided with this patch.  This patch 
 addresses template content redesign and defect corrections to the C&P 
 examination templates accessible through CAPRI CPWM.
  
 The following defects are being addressed in the patch for Spine and 
 Muscles:
  
 Spine
 =====
 Values for Pain (Pinprick) lower extremities is reversed.
 To recreate:
  
 PE tab
 Neuro Exam button
 Sensory Button
 **Scroll down to "LOWER EXTREMITIES"
 Left Pain (Pinprick) = 0
 Right Pain (Pinprick) = NT
 Preview report and the values are opposite
  
 Muscles
 =======
  
 Group 3 muscles injured: unable to select either Pectoralis Major 
 (Clavicular Head) or Deltoid---defaults to none.
  
 To recreate:
  
 Physical exam tab 
 HAS A MUSCLE BEEN INJURED, DESTROYED, OR TRAVERSED? 
 Answer: Yes
 Pop up box----select Shoulder Girdle and Arm
 Enter "none" for groups 1 & 2 
 Group 3 muscles injured: unable to select either Pectoralis Major 
 (Clavicular Head) or Deltoid---defaults to none.

 3.  Select Installation Option: 
     ---------------------------
     NOTE:  The following are OPTIONAL - (When prompted for the INSTALL
            NAME, enter DVBA*2.7*115):
       
     a.  Backup a Transport Global - This option will create a backup
         message of any routines exported with this patch. It will not
         backup any other changes such as DD's or templates.
     b.  Compare Transport Global to Current System - This option will
         allow you to view all changes that will be made when this patch
         is installed.  It compares all components of this patch
         (routines, DD's, templates, etc.).
