Subject: SCHEDULING ENROLLMENT INFORMATION WITH SELECT PATIENT

Category: 
  - Routine

Description:
============

 This patch will correct the problem associated with selecting
 Action PT - Change Patient from the SDAM template and
 the display of enrollment scrolls off screen when being asked
 for user verification.
   
 ASSOCIATED NOIS:
 ===============
 CTX-1104-70683  Enrollment information scrolls off screen when
                 using PT action.

 ================
 N/A

 El Paso,TX
 Phoenix,AZ
 Salisbury,NC
  
 NOIS OVERVIEW:
 ==============
  
 CTX-1104-70683  Enrollment information scrolls off screen when
                 using PT action.
 Problem:
 -------
 
 When selecting action PT (Change Patient) on SDAM template the
 subsequent patient enrollment display scrolls off screen without
 user verification (OK?  Yes//).
   
 Resolution:
 -----------
  
 Modify routine SDAM10 to pause for user verification of displayed patient
 enrollment information.

 ROUTINE CHANGES:
 ===============
 Routine:  SDAM10
  
 Add:
      PAT1    S %=1 W !,"   ...OK" D YN^DICN I %=0 W "   Answer
                 With 'Yes' or 'No'" G PAT1
              I %'=1 S Y=-1
