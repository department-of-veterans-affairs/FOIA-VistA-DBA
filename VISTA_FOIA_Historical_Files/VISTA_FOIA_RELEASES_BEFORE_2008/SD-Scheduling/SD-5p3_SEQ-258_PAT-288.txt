Subject: SD WAIT PARAMETER FILE FIX

Category: 
  - Routine

Description: 
============

 This patch addresses the issue of entries in file SD WL CLINIC LOCATION 
 (#409.32) and the field INSTITUTION (#.02), which points to 
 the INSTITUTION (#4) file.  A screen is added to the input routine to 
 allow only clinics that point to national institutions and are medical
 treating facilities to be added to file SD WL CLINIC LOCATION (#409.32).
  
 This patch addresses the following E3R(s):
 ------------------------------------------
 There are no E3Rs associated with this patch.
  
 This patch addresses the following NOIS message(s):
 ---------------------------------------------------
 BIR-0303-30542
 NIN-0303-40556
  
 Overview of NOIS Message(s):
 --------------------------
 BIR-0303-30542
 NIN-0303-40556
  
 Problem:
 ________
  
 Clinics with Local Institutions can be added to SD WL CLINIC LOCATION 
 (#409.32) file and Clinics with National Institutions are prevented from
 being entered.
  
 Resolution:
 ___________
  
 Edit routine SDWLPE to screen for local institutions and non-treating 
 facilities. Also allow additions for Clinics with National Institutions.
