 All DSM, Cache for Windows NT and Cache for OpenVMS sites are instructed 
 to install patch XU*8.0*268 followed by patch KMPS*1.8*3
  
 This patch corrects problems with Statistical Analysis of Global Growth 
 (SAGG) package for the Cache for OpenVMS implementation.
  
 This patch will add the creation date to the ^XTMP("KMPS",0) global to
 make the node SACC compliant for all M implementation platforms.

 Installation Instructions:
 ==========================
  
 This patch can be installed with VISTA online. Provided that the 'SAGG
 Master Background Task' [KMPS SAGG REPORT] is not currently running, this
 installation can be performed at any time of the day with minimal
 disruption to users. If desired, this patch can be Queued for a time of 
 generally least activity.
  
 Users may remain on the system during patch installation.
  
 TaskMan does not need to be STOPPED or placed in a WAIT state.
  
 Patch installation should not take longer than 15-30 minutes.
  
    1.  Ensure that the 'SAGG Master Background Task' [KMPS SAGG REPORT] 
        is not currently running by using the 'Check SAGG Environment' 
        [KMPS SAGG STATUS] option to review the 'Current Status' field.
  
    2.  DSM sites - You will need to disable any routines within the
        %ZOSVKS* namespace.  
  
    3.  Download the KIDS file XU_8_268.KID from the ANONYMOUS.SOFTWARE
        directory of either Albany, Hines or the Salt Lake OIFO to the
        appropriate directory on your system.
  
    4.  Use 'Load a Distribution' option on the KIDS 'Installation' menu,
        and enter: XU_8_268.KID
  
        NOTE: Provide the necessary directory path to the file if needed.
  
    5.  The patch has now been loaded into a transport global on your 
        system.  You now need to use KIDS to install the transport global.
  
    6.  On the KIDS menu, under the 'Installation' menu, use the following 
        options: 
  
           Verify Checksums in Transport Global 
           Print Transport Global 
           Compare Transport Global to Current System 
           Backup a Transport Global 
  
    7.  On the KIDS menu, under the 'Installation' menu, use the 'Install 
        Package(s)' option:
        
           Select INSTALL NAME: XU*8.0*268
                                ==========
  
    8.  Answer 'NO' to 'Want KIDS to INHIBIT LOGONs during the install?'
        Answer 'NO' to 'Want to DISABLE Scheduled Options, Menu Options,
        and Protocols?'
  
    9.  Cache sites - you do not have to move any routines, skip to
        Step 11.
  
   10.  DSM sites - you need to move several routines from your production 
        account to manager account.
  
           ZOSVKSD
           ZOSVKSOE
           ZOSVKSOS
           ZOSVKSVE
           ZOSVKSVS
  
   11.  Cache sites - remain in VAH and perform the instruction in Step 13.
  
   12.  DSM sites - remain in the manager account and perform the 
        instruction in Step 13.
  
   13.  ALL sites - perform the following instruction.
        
        > D RELOAD^ZTMGRSET
          Select the System
          Patch number to load: 268
                                ===
  
   14.  DSM sites - after the patch is installed, rebuild your map set if
        necessary.
  
        The %ZOSVKS* namespaced routines are NOT recommended to be mapped.
  
   15.  All Sites MUST continue with the installation instructions provided
        with patch KMPS*1.8*3.
