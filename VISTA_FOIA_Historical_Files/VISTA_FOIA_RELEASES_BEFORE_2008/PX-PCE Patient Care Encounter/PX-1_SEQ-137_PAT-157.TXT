Subject: EDUCATION TOPICS: TEXT EDITS NEEDED/TEXT DISPLAY PROBLEM

Category: 
  - Print Template
  - Routine

Description:
============

 1.  Will edit Education Topics: VA-ADVANCE DIRECTIVES and 
     VA-ADVANCED DIRECTIVES SCREENING to insure term "Advance directives"
     is used when necessary in lieu of various alliterations found.
  
 2.  Corrects problem entering outlined or longer sentences in Education
     Topic Add/Edit [PXTT EDIT EDUCATION TOPICS] option for Outcomes and
     Standards and then inquiring through Education Topic Inquiry [PXTT
     INQUIRE EDUC TOPIC] option causing invalid line wrapping.
  
  
 Associated E3R(s):
 =================
  
   
 Remedy:
 ======
 HD64732  PUG-0105-51304 Text edits needed for wording "Advance Directives"
                         in Education Topics: VA-ADVANCE DIRECTIVES and
                         VA-ADVANCED DIRECTIVES SCREENING.
  
 HD64368  PUG-1104-50142 Wrap problem inquiring for Education Topic
                         for GOALS and STANDARDS text.
  
   
  
 PARTICIPATING TEST SITES:
 ========================

 Charleston, SC 
  
   
 Remedy Overview:
 ===============
 1. Text edits needed for Education Topics: VA-ADVANCE DIRECTIVES and
    VA-ADVANCED DIRECTIVES SCREENING.
  
 Problem:
 =======
 Text editing needed for Education Topics: VA-Advance Directives and
 VA-Advanced Directives Screening for proper wording. 
   
 Resolution:
 ==========
 Post install routine PX157PST created to alter all alliterations of
 Advance Directives to wording "Advance Directives" for Education Topics:
 VA-Advance Directives and VA-Advanced Directives Screening.
  
  
  
 2. Wrap problem inquiring on GOALS and STANDARDS for Education Topics
 when using Education Topic Inquiry [PXTT INQUIRE EDUC TOPIC] option. 
  
 Problem:
 =======
   
 During edit in Education Topic Inquiry [PXTT INQUIRE EDUC TOPIC] option, 
 the Educational Outcomes and Standards are entered through a screen editor
 in various site formats.  Wrapping problems occur during Education Topic 
 Inquiry [PXTT INQUIRE EDUC TOPIC] option due to GOALS and STANDARDS line
 headings and text indentation.
    
 Resolution:
 ------------
   
 Modification made to the Print Template PXTT EDUCATION TOPIC DETAIL
 to print GOALS and STANDARD line descriptors on separate line.
  
  
  
 INSTALLATION INSTRUCTIONS
 =========================
  
 This patch can be installed at any time.  Users do not need to be off
 the system.  Installation will take 5 minutes or less.
  
 1.  Choose the PackMan message containing this patch and invoke the
     INSTALL/CHECK MESSAGE PackMan option.
   
 2.  Start up the Kernel Installation and Distribution System Menu
     [XPD MAIN]: 
     
           Edits and Distribution ...  
           Utilities ...  
           Installation ...  
     
     Select Kernel Installation & Distribution System Option: INStallation
                                                              ---
                                                                 
           Load a Distribution 
           Print Transport Global 
           Compare Transport Global to Current System 
           Verify Checksums in Transport Global 
           Install Package(s) 
           Restart Install of Package(s) 
           Unload a Distribution 
           Backup a Transport Global  
   
 3.  From this menu, you may elect to use the following options
     (when prompted for the INSTALL NAME, enter PX*1.0*157):
   
      a.  Backup a Transport Global - This option will create a backup
          message of any routines exported with this patch. It will not
          backup any other changes such as DD's or templates.
      b.  Compare Transport Global to Current System - This option will
          allow you to view all changes that will be made when this patch
          is installed.  It compares all components of this patch
          (routines, DD's, templates, etc.).
      c.  Verify Checksums in Transport Global - This option will allow
          you to ensure the integrity of the routines that are in the
          transport global.
  
 4.  Use the Install Package(s) option and select the package PX*1.0*157
   
      a.  When prompted 'Want KIDS to INHIBIT LOGONs during the install?
          YES//', answer NO (unless otherwise indicated).
      b.  When prompted 'Want to DISABLE Scheduled Options, Menu Options,
          and Protocols? YES//', answer YES.  When prompted to select the
          options you would like to place out of order, enter:
  
          PXTT INQUIRE EDUC TOPIC      Education Topic Inquiry
          PXTT EDIT EDUCATION TOPICS   Education Topic Add/Edit
  
      c.  When prompted 'Enter protocols you wish to mark as 'Out Of 
          Order':', press <return>.
  
 5.  There is an associated post-install routine, PX157PST, that will send 
 a mail message to the installer upon completion, approximately 10 seconds 
 after the installation.
