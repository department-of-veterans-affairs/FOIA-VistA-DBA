Subject: MFS/SDS/MD5

Category: 
  - Routine
  - Data Dictionary

Description:
============

 This patch includes MFS enhancements to support Standard Data Service (SDS)
 update message types.  It also includes MD5 functionality.
  
 The patch adds the MD5 Signature and Standard Terminology Version file
 parameters to MASTER FILE PARAMETERS (#4.001).

 NOTE:
  
 This patch adds file/sub-file numbers to the MD5 Signature file for
 non-Kernel files like the Allergy and Vitals files.  If your VistA
 instance is a non-medical center (example: CMOP), you may see the
 following:
  
   could not be added to file 4.0051.
  
   Record with .01 value 120.5399 and internal entry #120.5399
   could not be added to file 4.0051.
  
 You may ignore this message.  Your patch installed correctly.
