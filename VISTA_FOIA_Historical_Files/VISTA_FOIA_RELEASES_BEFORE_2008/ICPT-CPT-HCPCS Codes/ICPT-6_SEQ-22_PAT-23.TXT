Subject: Code Set FY05 3rd Qtr Update

Category: 
  - Informational

Description:
============

  This patch is an informational patch announcing the release of the 
  CPT/HCPCS Modifier changes for FY 2005 effective Apr 1, 2005.  This
  patch contains no routines, data, or data dictionaries.  The data 
  for this patch is exported with the corresponding Lexicon data in
  patch LEX*2.0*35.  Patch LEX*2.0*35 requires that patch LEX*2.0*34
  be installed first.
  Note: This patch also includes codes that have an effective date
  earlier than Apr 1, 2005. These are retroactive changes that have
  been released by the Standards Development Organization this quarter.
  
  The data in LEX*2.0*35 affects the following CPT/HCPCS files: 
  
          81         CPT
          81.3       CPT MODIFIER
