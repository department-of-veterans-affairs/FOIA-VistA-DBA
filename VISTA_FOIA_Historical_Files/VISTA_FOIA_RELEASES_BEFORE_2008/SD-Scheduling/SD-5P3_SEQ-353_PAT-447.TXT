Subject: IEMM ERROR TABLE UPDATE

Category: 
  - Informational

Description:
============

 This is an informational patch. The Incomplete Encounter 
 Management Module (IEMM) Error Table is being updated 
 on the Vista Document Library (VDL).
  
 ASSOCIATED REMEDY TICKET:
 =========================
 1. HD64514 - Requesting IEMM Error Table Update
  
 ASSOCIATED E3R(S):
 ==================
 N/A
  
 REMEDY TICKET OVERVIEW:
 =======================
 1. HD64514 - Requesting IEMM Error Table Update
  
    Problem:
    --------
    It has been requested that the IEMM Error Table located on the VDL
    be updated.  Additionally, it has been requested that the Release
    Notes Appendix - IEMM Error Table, currently found as an Appendix
    to Release Notes - Incomplete Encounter Mgmt Module (IEMM), be
    relabeled as an Appendix to the Ambulatory Care Reporting Project.
  
    Resolution:
    -----------
    35 new 3-digit NPCD error codes and 30 new 4 digit Vista error
    codes will be added to the existing IEMM Error Table document.
  
    The new error codes are:
  
    NPCD  VISTA   ERROR DESCRIPTION
    ----  -----   -----------------
    238   2380    Ethnicity code missing or invalid
    240   2400    Conf. Address - Line 1 contains all numbers
    241   2410    Conf. Address - Line 2 contains all numbers
    242   2420    Conf. Address City is missing or invalid
    243   2430    Conf. Address State code is missing or invalid
    244   2440    Conf. Address Zip Code is missing or invalid
    245   2450    Conf. Address County code is invalid
    246           Conf. Address Inactive State/County Code
    247   2470    Conf. Address Category is missing or invalid
    248   2480    Conf. Address Start Date is missing or invalid
          3030    The encounter date is greater than 14 days after the 
                     date of death.
    325           Prisoner of War Location is invalid
          3250    Prisoner of War Location is invalid or inconsistent with
                     POW status.
    326           Prisoner of War Location is inactive
    328   3280    Prisoner of War Location inconsistent with POW status
    630   6300    CPT modifier invalid

    637   6370    CPT modifier coding method is missing or invalid
    704   7040    MST Status Invalid
    706   7060    MST status date invalid or inconsistent with MST status
    715   7150    Radiation Exposure Method is invalid or inactive

    Also, the Release Notes Appendix - IEMM Error Table document will be
    relabeled as an Appendix to the Ambulatory Care Reporting Menu, as
    follows:
