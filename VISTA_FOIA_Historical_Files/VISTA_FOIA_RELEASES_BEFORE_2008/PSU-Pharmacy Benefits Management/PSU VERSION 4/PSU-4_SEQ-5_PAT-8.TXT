Subject: INCORRECT DATA SHOWS UP ON PROVIDER SUMMARY REPORT

Category: 
  - Routine

Description:
============

 This patch will address the following three issues:
  
 Issue #1: The Provider Summary report is reporting missing data.  When a
 Provider has a Service/Section, the Provider will incorrectly show up on
 the report as missing specific data.
  
 Issue #2:  When running Manual Pharmacy Statistics [PSU PBM MANUAL] 
 option, the clinic that appears in a mailman message is not
 the same clinic that appears in the PRESCRIPTION file (#52).
  
 Issue #3: This patch will clean up routines no longer being used in the 
 Pharmacy Benefits Management package.
  
  
 Remedy Overview:
 ================
 I. REMEDY HD67718 & HD 131897
  
 Problem:
 The Provider Summary report is reporting missing data.  When a Provider 
 has a Service/Section other than that listed below, the Provider will 
 incorrectly show up on the report as missing specific data.
  
 AMBU AMB Ambulatory Care
 PRIMARY AMB Ambulatory Care
 CBOC AMB Ambulatory Care
 ANESTH ANES Anesthesiology
 CARDIO CV Cardiology
 PHARM CPHAR Clinical Pharmacy
 DENT DDS Dental
 INTERMED IM Intermediate Medicine
 MEDIC MED Medicine
 NEUROL NEUR Neurology
 NUCLEAR NUM Nuclear Medicine
 NURSING RN Nursing
 OPHTH OPH Ophthalmology
 ORTHOPED ORTHO Orthopedics
 PSYCHIA PSY Psychiatry
 MENTAL PSY Psychiatry
 PULM PUL Pulmonary
 RADIOL RAD Radiology
 SURG SUR Surgery
 UROLOG U Urology
  
 Solution:
 This patch will correct the above report.  Providers who have ANY 
 Service/Section will not show on the Provider Summary report.
  
 II. REMEDY 132806
 Problem with outpatient clinic.  When running the Manual Pharmacy
 Statistics [PSU PBM MANUAL] option, the clinic that appears in a mailman
 message is not the same clinic that appears in the PRESCRIPTION file 
 (#52)
  
  
 Solution:
 This patch will correct the above mailman message.  The Prescription 
 clinic will match the clinic in the PRESCRIPTION file (#52).

  Installation Instructions:
  ==========================
  It is recommended that this patch be installed after normal business
  hours while no Outpatient Pharmacy users are processing orders.
  
  Installation will take about 5 minutes.
  
    1. Use the INSTALL/CHECK MESSAGE option on the PackMan menu.
  
    2. From the Kernel Installation & Distribution System menu, select
       the Installation menu.
  
    3. From this menu, you may select to use the following options
       (when prompted for INSTALL NAME, enter PSU*4.0*8)
     a. Backup a Transport Global - this option will create a backup
        message of any routines exported with the patch. It will NOT
        backup any other changes such as DDs or templates.
     b. Compare Transport Global to Current System - this option will
        allow you to view all changes that will be made when the patch
        is installed. It compares all components of the patch (routines,
        DDs, templates, etc.).
     c. Verify Checksums in Transport Global - this option will ensure
        the integrity of the routines that are in the transport global.
  
    4. Use the Install Package(s) option and select the package PSU*4.0*8
  
    5. When prompted "Want KIDS to INHIBIT LOGONs during the install?
       YES//" respond NO.
  
    6. When prompted "Want to DISABLE Scheduled Options, Menu Options, and
       Protocols? YES//" respond NO.
