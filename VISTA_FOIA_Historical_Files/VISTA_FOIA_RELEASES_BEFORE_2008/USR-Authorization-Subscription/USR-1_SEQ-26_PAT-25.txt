Subject: Update for User Class Membership

Category: 
  - Routine

Description:
============

 This patch updates the evaluation process for User Class Membership when 
 determining co-signature requirements for TIU documents.
 
 Co-signature requirements will now be determined based on the ENTRY 
 DATE/TIME of the TIU Document instead of the current date.
 
 NOIS/E3R:
 ========
 PUG-1101-52116 has been maintained as a High Priority E3R and is addressed, 
 in part, by this patch.  TIU*1.0*175 must also be installed to fully 
 satisfy this E3R.

 Installation Instructions: 
 =========================
 This patch may be installed with users on the system.
 Installation of this patch should take less than 1 minute.
  
 1.  Use the INSTALL/CHECK MESSAGE option on the Packman Menu.  
  
 2.  From the Kernel Installation and Distribution System (KIDS) Menu, 
     select the Installation menu.  
  
 3.  From the Installation menu, you may elect to use the following 
     options (when prompted for INSTALL NAME, enter USR*1.0*25): 
  
     a.  Backup a Transport Global 
     b.  Compare Transport Global to Current System 
     c.  Verify Checksums in Transport Global
  
 4.  Use the Install Package(s) option and select the package USR*1.0*25.
  
 5.  When prompted 'Want KIDS to INHIBIT LOGONs during the install? YES//',
     respond NO.
  
 6.  When prompted 'Want to DISABLE Scheduled Options, Menu Options, and 
     Protocols? YES//', respond NO.
