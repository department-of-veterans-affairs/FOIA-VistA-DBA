Description:
============

  This patch is part of a multi-package Kernel Installation and Distribution
  (KIDS) distribution containing the following five patches installed in the
  order listed below:
 
    LEX*2.0*52   Fixes for Lexicon, ICD and CPT problems reported on Remedy
    ICPT*6.0*37  Routine ICPTMOD and ICPTMOD2
    ICD*18.0*29  Routine ICPTCODE and DD changes to the Diagnosis file #80
    LEX*2.0*53   FY08 1st quarter ICD/CPT update
    ICPT*6.0*38  Informational patch installed with LEX*2.0*53
 
  This multi-package distribution writes to the following files:
