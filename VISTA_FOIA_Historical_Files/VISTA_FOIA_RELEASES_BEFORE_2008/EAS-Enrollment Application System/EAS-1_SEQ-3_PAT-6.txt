Subject: MISSING COMPONENTS IN EAS V1.0

Category: 
  - Other

Description: 
============

   Patch EAS*1*6 distributes one List Template and two Protocols
   that were mistakenly overlooked in the original distribution of
   EAS v1.0.
   
   The List Template and Protocols distributed in this patch are
   needed to support viewing of 'Inactivated/Closed' 1010EZ 
   Applications within the List Manager screens of the EASEZ module.
   
   
  TECHNICAL DESCRIPTION:
  ======================
               
  There is one NOIS report associated with this patch:
   
     MAC-0801-62048 
    
  There is one List Template (file #409.61) distributed with this patch:
    
     EAS EZ 1010EZ REVIEW6
   
  There are two Protocols (file #101) distributed with this patch:
   
     EAS EZ 1010EZ REVIEW6
     EAS EZ QUIT
    
  There are no routines distributed with this patch.
   
      
  INSTALLATION:
  =============
                
  Install Time - less than 2 minutes                        
                           
  ======================= INSTALLATION INSTRUCTIONS =======================      
                 
    
  1.   LOAD TRANSPORT GLOBAL 
       ---------------------
       Choose the PackMan message containing this patch and invoke the 
       INSTALL/CHECK MESSAGE PackMan option to load the patch contents
       to the KIDS transport global.  
                              
  2.   START UP KIDS 
       -------------
       Start up the Kernel Installation and Distribution System Menu 
       [XPD MAIN]: 
                          
            Edits and Distribution ...  
            Utilities ...  
            Installation ...  
                                    
       Select Kernel Installation & Distribution System Option: INStallation 
                                                                ---
            Load a Distribution 
            Print Transport Global 
            Compare Transport Global to Current System 
            Verify Checksums in Transport Global 
            Install Package(s) 
            Restart Install of Package(s) 
            Unload a Distribution 
            Backup a Transport Global 
                      
       Select Installation Option: 
    
       NOTE:  The following are OPTIONAL - 
                     
       a.  Backup a Transport Global - This option will create a backup
           message of any routines exported with this patch. It will not
           backup any other changes such as DD's or templates.
       b.  Compare Transport Global to Current System - This option will
           allow you to view all changes that will be made when this patch
           is installed.  It compares all components of this patch
           (routines, DD's, templates, etc.).
       c.  Verify Checksums in Transport Global - This option will allow
           you to ensure the integrity of the routines that are in the
           transport global.        
           
  3.   SELECT INSTALLATION OPTION: 
       --------------------------
                    
       **This is the step to start the installation of this KIDS patch. 
     
       Select Installation Option: Install Package(s) 
                                   ----------------
       Select INSTALL NAME: EAS*1.0*6
                            ---------
    
       a.  Choose the Install Package(s) option to start the patch install.
       b.  When prompted for the INSTALL NAME, enter EAS*1.0*6.      
       c.  When prompted 'Want to DISABLE Scheduled Options, Menu 
           Options, and Protocols? YES//' answer NO.
