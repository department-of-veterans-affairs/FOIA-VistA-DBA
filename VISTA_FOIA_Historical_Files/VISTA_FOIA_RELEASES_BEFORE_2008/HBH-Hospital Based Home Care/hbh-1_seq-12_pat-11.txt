Description: 
============

   1)  New report & option:  Total Visits by Date Range Report  (80)
       [HBHCRP21] 
 
       This option prints HBPC Total Visits by Date Range Report.  User is  
       prompted to enter date range for inclusion on report.  Visit is      
       omitted from report if contains any of the following CPT codes: 
  
         99358  PROLONGED SERV, W/O CONTACT
         99359  PROLONGED SERV, W/O CONTACT
         99361  PHYSICIAN/TEAM CONFERENCE
         99362  PHYSICIAN/TEAM CONFERENCE
         99371  PHYSICIAN PHONE CONSULTATION
         99372  PHYSICIAN PHONE CONSULTATION
         99373  PHYSICIAN PHONE CONSULTATION
         99374  HOME HEALTH CARE SUPERVISION
         99375  HOME HEALTH CARE SUPERVISION
         99376  CARE PLAN OVERSIGHT/OVER 60
 
       User can also select additional CPT codes to omit from report. 
       Report includes:  Patient Name, SSN Last Four, Total Visits per
       patient, Date  (admitted to HBPC program) & Discharge Date (if
       applicable), with grand totals of patients & visits.  Report prints
       in 80 column format. 
 
   Solution/Routine(s) changed:  HBHCRP21 routine & option
 
 
   2)  New report & option:  Provider CPT Code Summary Report  (80) 
       [HBHCRP22] 
 
       This option prints HBPC Provider CPT Code Summary Report.  User is
       prompted to enter date range, CPT code(s) (can be range of CPTs), &
       Provider for inclusion on report.  Report includes:  Provider Name
       in header, CPT code(s) with summary total(s) & grand total for all
       CPTs.  Report prints in 80 column format. 
 
   Solution/Routine(s) changed:  HBHCRP22 routine & option, HBHCUTL4 routine
 
 
   3)  Routine HBHCRP17, which produces CPT Code Summary Report  (80)
       [HBHCRP17], was revamped to utilize HBHCUTL4, utility routine module,
       to avoid code duplication for HBHCRP22 option 
 
   Solution/Routine(s) changed:  HBHCRP17, HBHCUTL4 routines
 
 
   4)  Changed W *7 to W $C(7) to ensure portability
 
   Solution/Routine(s) changed:  HBHCUTL4 routine

    Patch Installation:
    ==================
  
    Only HBPC  (Home Based Primary Care, formerly HBHC Hospital Based Home
    Care)  users need to be off the system.  Patch installation time should
    be minimal.  There is no recommended time of day for installation. 
  
    1.  Review your mapped set & remove any routines listed in ROUTINE      
        SUMMARY section. 
  
    2.  Use INSTALL/CHECK MESSAGE option on PackMan menu.
  
    3.  From Kernel Installation and Distribution System Menu, select
        Installation.
  
    4.  From Installation menu, you may elect to use the following options,
        select INSTALL NAME  HBH*1.0*11  
  
          a.  Verify Checksums in Transport Global
    
          b.  Compare Transport Global to Current System
 
          c.  Backup a Transport Global.
 
    5.  Use Install Package(s), select package  HBH*1.0*11.
  
    6.  Answer NO to the prompt Want to DISABLE Scheduled Options, Menu
        Options, and Protocols. 
  
    7.  Return any routines unmapped in step 1, to the mapped set once
        installation is complete.
