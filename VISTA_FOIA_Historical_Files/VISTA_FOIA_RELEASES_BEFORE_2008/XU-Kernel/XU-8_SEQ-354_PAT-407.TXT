Subject: MFS generic message w/ TIU and Orders parameters

Category: 
  - Routine
  - Data Dictionary

Description:
============

 This patch enhances the Master File Server (MFS) mechanism
 by handling a generic HL7 message structure developed together
 with the Standards and Terminology Services (STS) team.  The
 patch also distributes TIU and Orders parameters.
  
 To receive HL7 messages VistA HL7 version 1.6 is required.  Enter a Remedy
 ticket for the HSITES Infrastructure Team if you need a complete build of
 the HL7 package.
  
 If installing this patch in a test account, or an account that does not 
 require Data Standardization updates, then you may install the patch 
 without HL7 messaging installed and/or running.
  
 The following components are provided:
 --------------------------------------
  
   PROTOCOLS:
      XUMF MD5 EVENT        MD5 checksum request
      XUMF MD5 HANDLER      MD5 checksum request handler
      XUMF QUERY EVENT      MFS data request
      XUMF QUERY HANDLER    MFS data request handler
      XUMF UPDATE           MFS update handler
      XUMF VETS UPDATE      MFS push/update
  
   HL7 APPLICATION PARAMETERS:
      VETS DATA             MFS data pull
      VETS MD5              MD5 checksum request
      VETS UPDATE           MFS push/update
      XUMF DATA             MFS data request handler
      XUMF MD5              MD5 checksum request handler
      XUMF UPDATE           MFS update handler
