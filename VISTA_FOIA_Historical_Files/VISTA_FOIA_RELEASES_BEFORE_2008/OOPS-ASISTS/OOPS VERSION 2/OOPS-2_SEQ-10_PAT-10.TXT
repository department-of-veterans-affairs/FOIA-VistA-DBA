Subject: DECOMMISSION ROLL AND SCROLL MENU OPTIONS

Category: 
  - Routine

Description:
============

 This patch will decommission designated Automated Safety
 Incident Surveillance and Tracking System (ASISTS) roll and
 scroll menu options to insure VHA facilities are in compliance
 with the Deputy Under Secretary of Operations and Management's
 Memorandum dated March 19, 2003.

 REMEDY TICKET OVERVIEW:
 =======================
 HD0000000059133   REQUEST FOR ASSISTANCE
  
    Problem:
    --------
    Deputy Under Secretary of Operations and Management's
    Memorandum, dated March 19, 2003, directed all VHA
    facilities to implement the ASISTS Graphical User Interface
    (GUI).  Additionally, designated ASISTS roll and scroll
    menu items were required to be placed "Out of Order" at
    all VHA facilites.
  
  
    Resolution:
    -----------
    This patch will use the KIDS Build to decommission (or delete)
    the following forty-six ASISTS roll and scroll menu options.
    Ten ASISTS roll and scroll menu options will remain in use
    following this patch installation.
  
    OOPS CREATE AMENDMENT (SO)  Create Amendment
    OOPS CREATE AMENDMENT (SUP) Create Amendment
    OOPS CREATE CASE            Create Accident/Illness Record
    OOPS EDIT SITE PARAMETER    Edit Site Parameter
    OOPS EMP DISPLAY CA1/CA2    Display CA1 or CA2
    OOPS EMP ENTRY              Complete Employee CA1/CA2
    OOPS EMP HEALTH MENU        ASISTS Employee Health Menu
    OOPS EMP HLT DISP CA1/CA2   Display CA1/CA2
    OOPS EMP HLT EDIT STUB      Edit Stub Record
    OOPS EMP HLT WCP EMP SIGN   Approve Worker's Comp Signing for Employee
    OOPS EMP MENU               ASISTS Employee Menu
    OOPS EMP PRINT CA           Print CA1/CA2
    OOPS EMP VALIDATE           Validate and Sign CA-1 or CA-2
    OOPS FIX PAY RATE PER FIELD Fix PAY RATE PER Field
    OOPS LOG   Log of Federal   Occupational Injuries and Illnesses
    OOPS NEEDLESTICK LOG        Log of Needlestick Incidents
    OOPS PRINT BILL             Print Employee Bill of Rights
    OOPS SAFETY CLOSE           Change the Status of a Case
    OOPS SAFETY DISPLAY CA1/CA2  Display CA1 or CA2
    OOPS SAFETY EDIT STUB RECORD Edit Stub Record
    OOPS SAFETY ENTER 2162      Complete Report of Accident (2162)
    OOPS SAFETY MENU            ASISTS Safety Officers Menu
    OOPS SAFETY PRINT           Print Report of Accident
    OOPS SAFETY PRINT CA        Print CA1/CA2
    OOPS SAFETY PRINT STATUS    Print Accident Report Status
    OOPS SAFETY SUP ENTRY       Edit Report of Incident
    OOPS SAFETY VALIDATE        Validate and Sign 2162
    OOPS SAFETY WCP EMP SIGN    Approve Worker's Comp Signing for Employee
    OOPS SUP DISPLAY CA1/CA2    Display CA1 or CA2
    OOPS SUP EMP ENTRY          Complete Employee CA-1 & CA-2
    OOPS SUP ENTRY              Edit Report of Incident
    OOPS SUP MENU               ASISTS Supervisor Menu
    OOPS SUP PRINT              Print Report of Accident
    OOPS SUP PRINT CA           Print CA1/CA2
    OOPS SUP PRINT STATUS       Print Accident Report Status
    OOPS SUP VALIDATE           Validate and Sign CA1, CA2 or 2162
    OOPS UNION LOG              Log of Federal Occupational 
                                Injuries and Illnesses
    OOPS UNION MENU             ASISTS Union Menu
    OOPS UNION PRINT            Print Report of Accident
    OOPS UNION PRINT STATUS     Print Accident Report Status
    OOPS WC EDIT CA1/CA2        Worker's Compensation Edit CA1/CA2
    OOPS WC EDIT UNION INFO     Enter/Edit Union Information
    OOPS WC EMP ENTRY           Complete Employee CA-1 & CA-2
    OOPS WC SIGN FOR EMPLOYEE   Worker's Comp Elect. Sign for Employee
    OOPS WC VALIDATE            Validate and Sign CA1 or CA2
    OOPS WORKER'S COMP MENU     ASISTS Worker's Compensation Menu
  
    The following ten ASISTS roll and scroll menu options will remain
    in use at all VHA facilities:
  
    OOPS DOL MANUAL XMIT DATA           Manual Transmission of DOL Data
    OOPS DOL SCHEDULED XMIT DATA        Scheduled Transmission of DOL Data
    OOPS GUI EMPLOYEE   ASISTS GUI      Employee Menu (Context)
    OOPS GUI EMPLOYEE HEALTH MENU       ASISTS GUI Employee Health Menu
    OOPS GUI SAFETY OFFICER MENU        ASISTS GUI Safety Menu (Context)
    OOPS GUI SUPERVISOR MENU            ASISTS GUI Supervisor Menu (Context)
    OOPS GUI UNION MENU                 ASISTS GUI Union Menu (Context)

 INSTALLATION INSTRUCTIONS:
 ==========================
 Users can be on the system during installation of this patch.
  
 Installation should take less than 10 minutes.
  
 1.  LOAD TRANSPORT GLOBAL
     ---------------------
     Choose the PackMan message containing this patch and invoke the
     INSTALL/CHECK MESSAGE PackMan option.
  
 2.  DISABLE ROUTINE MAPPING  (DSM for Open VMS sites only)
     -----------------------
     Does not apply to this patch as there are no routines involved.
  
 3.  START UP KIDS
     -------------
     Start up the Kernel Installation and Distribution System Menu
     [XPD MAIN]
  
           Edits and Distribution ...
           Utilities ...
           Installation ...
  
     Select Kernel Installation & Distribution System Option:  INStallation
                                                               ---
  
     1     Load a Distribution
     2     Verify Checksums in Transport Global
     3     Print Transport Global
     4     Compare Transport Global to Current System
     5     Backup a Transport Global
     6     Install Package(s)
           Restart Install of Package(s)
           Unload a Distribution
  
 4.  From this menu, you may elect to use the following options
     (when prompted for INSTALL NAME, enter OOPS*2.0*10):
  
     a.  Print Transport Global - this option will allow you to view the
        components of the KIDS build.
  
     b. Backup a Transport Global - This option will create a backup
         message of any routines exported with the patch.  It will NOT
         backup any other changes such as DDs or templates.
  
     c.  Compare Transport Global to Current System - This option will
         allow you to view all changes that will be made when the patch
         is installed.  It compares all components of the patch
         (routines, DDs, templates, etc.).
  
 5.  Use the Install Package(s) option and select the package OOPS*2.0*10.
  
     a.  When prompted "Want KIDS to rebuild Menu Trees Upon Completion
         of Install? YES//" respond NO.
  
     b.  When prompted "Want KIDS to INHIBIT LOGONS during the
         install? YES//" respond NO.
  
     c.  When prompted "Want to DISABLE Scheduled Options, Menu
         Options, and Protocols? YES//" respond NO.
  
 6.  BUILD MAPPED ROUTINE(S)   (DSM for Open VMS sites only)
     -----------------------
     N/A
