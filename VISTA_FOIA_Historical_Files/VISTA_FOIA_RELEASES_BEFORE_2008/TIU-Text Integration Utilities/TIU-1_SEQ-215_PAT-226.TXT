Subject: PROBLEM WITH MEDICATION OBJECT

Category: 
  - Routine

Description:
============

 This patch addresses a defect reported in Remedy ticket 
 HD 183284 (PSI-07-063) where the medications object does not return the 
 Schedule for an inpatient IV.
  
 NOTE: In the Remedy ticket it is noted that the Reports Tab in CPRS
 does not display schedule.  This will be addressed in OERR patch,
 OR*3.0*243 V27.
  
 ASSOCIATED REMEDY:
 ==================
 HD183284  (PSI-07-063)
  
 PARTICIPATING TEST SITES:
 =========================
 Mountain Home, TN
 Tennessee Valley HCS
 Loma Linda, CA
 Madison, WI.
  
 PROBLEM:
 ========
 The medications object does not return the schedule for an Inpatient IV.
  
 RESOLUTION:
 ===========
 Modify routine TIULMED1 to return the route and schedule when the 
 medications object is executed.
  
 INSTALLATION INSTRUCTIONS:
 ==========================
 This patch can be loaded with users on the system but it is recommended 
 that it be scheduled for off hours. The medications object is embedded in
 a lot of documents and you could get an Edited error if loading during 
 peak activity. Installation time will vary but it should be less than 2
 minutes.
