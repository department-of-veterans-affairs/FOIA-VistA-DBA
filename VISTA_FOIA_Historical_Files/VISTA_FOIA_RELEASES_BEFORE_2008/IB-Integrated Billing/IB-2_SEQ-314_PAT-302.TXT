Subject: ADD NEW ADJUSTMENT CODE TO AUTO BILL GENERATION LIST

Category: 
  - Routine

Description:
============

 This patch adds CO 172 to the list of adjustment codes that meet
 auto-generation criteria to automatically create a Medicare Remittance
 Advice (MRA) secondary claim. 
  
  
 This patch addresses the following New Service Request (NSR):
 -------------------------------------------------------------
 There is no NSR associated with this patch.
  
  
 This patch addresses the following Remedy Ticket(s):
 ----------------------------------------------------
 1. HD136804     Add CO172 to adjustment codes that meet
                      auto-generation criteria  
  
  
 Overview of Remedy Ticket(s):
 -----------------------------
 1. HD136804    Add CO172 to adjustment codes that meet
                     auto-generation criteria
  
 Problem: 
 --------
 Due to the large number of these adjustment codes that need to be
 processed manually, it was decided that this adjustment code should be
 added to the list of codes that automatically trigger the creation of a
 MRA secondary claim. 
  
 Resolution:
 -----------
 Add CO 172 to the list of adjustment codes that are  automatically
 processed.
