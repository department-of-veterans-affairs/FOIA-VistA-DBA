Description: 
============

   
  This patch exports options and routines designed to help users
  create and manage the list of Health Summary Types listed on the
  Reports Tab of CPRS GUI.  It also contains several miscellaneous
  fixes to problems reported by NOISes or E3Rs.  The following is a
  list of the problems fixed by this patch:
    
    1   Renamed Radiology components as part of an         NOIS HUN-0800-21727
        effort to standardize national component           E3R 8045
        names across the VA.  (Routine GMTSP47, 
        Post-install)
    
          Renamed from:
    
              RADIOLOGY IMPRESSION                RI
              RADIOLOGY IMPRESSION SELECTED       SRI
              RADIOLOGY PROFILE                   RP
              RADIOLOGY STATUS                    RS
    
          To:
              IMAGING IMPRESSION                  II
              IMAGING IMPRESSION SELECTED         SII
              IMAGING PROFILE                     IP
              IMAGING STATUS                      IS
    
        NOTE:  This requires that OR*3.0*109 
               be installed first.
 
    2   All cancelled radiology exams will be clearly      NOIS ALN-0499-12690 
        marked with a status of 'Cancelled'.
        (Routine GMTSRAI)
    
    3   Included Case Number in the Imaging Impressions    NOIS IOW-0100-42187
        Health Summary component.  (Routine GMTSRAI)
    
    4   Modified the call to the data extract routine      NOIS BAN-0399-11156
        for the Current Orders Component to include
        both parent and children orders.  (Routine 
        GMTSORC3)
    
    5   Removed the misleading text "No Site" from         NOIS CHS-0399-40097
        Outpatient Encounter component.  (Routine
        GMTSPXOP)
    
    6   Added Attending and Ward to the ADT History        NOIS MIA-1199-32356
        Expanded (EADT) component.  (Routines 
        GMTSDGCH and GMTSDGC1)
    
    7   Modified the Print Template 'GMTS TYPE INQ' to     NOIS PUG-0800-52430
        allow a 4-character column for 'Time Limits'.
        Option 'GMTS TYPE INQ' is exported with two 
        supporting Print Templates, 'GMTS TYPE INQ 
        FOOTER' and 'GMTS TYPE INQ HEADER'.
  
    8   Removed references to the old Generic Progress     NOIS PUG-1200-51609
        Notes Package from the on-line help.  (Routine
        GMTSP47, Post-install)
    
    9   User is prompted for deletion of a Health          NOIS PUG-0800-52862
        Summary Type created without components.
        (Routines GMTSRM and GMTSRM2)
       
   10   Fixed the Health Summary Type page title for       NOIS NYN-0699-10707
        'Hospital Location Health Summary'. (Routines 
        GMTSPD and GMTSPL)
    
   11   Added Surgeon/Physician to the Surgical            NOIS POR-0699-50668
        Pathology (SP) component. (Routine GMTSLRA)
    
   12   Removed National Database Integration (NDBI)       NDBI FORUM #32463150
        modifications for the Laboratory components.
        (Routines GMTSLRA, GMTSLRC, GMTSLRCP, 
        GMTSLRM, GMTSLRMB, GMTSLRS, GMTSLRS7, 
        GMTSLRSC, GMTSLRSE, and GMTSLRT)
    
   13   Added a warning to Progress Notes components       NOIS DAN-0501-40841
        to inform the user when they are viewing 
        'RETRACTED' or 'DELETED' entries.  (Routines
        GMTSPN and GMTSPN2)
    
   14   After installing patch GMTS*2.7*45, the            NOIS MAR-0501-21605
        Brief Discharge Summary displayed the full
        Discharge Summary.  This has been fixed to 
        display the brief report. (Routines GMTSDS
        and GMTSDSB)
    
   15   Fixed undefined variable XMZ which resulted        NOIS SFC-0501-62115
        when a remote user accessed Mailman                     ANN-0601-41131
        functionality in the Medicine Health Summary            LOM-0501-61647
        component.  (Routine GMTSMCMA)                          LOM-0601-62249
                                                                GRJ-0701-50385
                                                                FNC-0801-32140
                                                                HOU-0801-71965
                                                                MAC-0701-62039
                                                                DAY-0501-42347
   
   16   Changed the display in Lab Orders component,       NOIS CPH-0601-41411
        increasing the display area for the accession
        number to 15 characters to prevent truncating
        the number.  (Routine GMTSLROS)
    
   17   Modified Health Summary Type lookup (file #142)    NOIS MWV-0601-20740
        so that duplicate Health Summary Types may not 
        be added to the file. (Routine GMTSULT)
    
   18   Fixed problem reported after installing            NOIS UNY-0601-11951
        OR*3.0*85, printing multi-page reports without
        headers and page numbers.  (Routine GMTSU)
    
   19   Fixed the date range problem for the Imaging       NOIS SUX-0501-42616
        Profile (IP) component to include the entire
        range of dates included in a visit using the
        Visit Patient Health Summary option. 
        (Routines GMTSRAS and GMTSRAE)
  
   20   Fix the undefined variable GMTSIEN in Radiology    NOIS PUG-0701-51885
        Component.  (Routine GMTSRAE)
   
   21   Fix the Medicine Components to extract data        NOIS UNY-0701-10956
        from the ENDOSCOPY/CONSULT file (#699) for
        endoscopy type data.  (Routine GMTSMCMA)
   
   22   Removed the 'LRSUPER' key from the Lab             NOIS CMO-0999-40426
        Surgical Pathology Health Summary Component.
    
   23   The Consults Brief component does not return       NOIS HUN-0701-20471
        data if the occurrence limit is not 
        used or deleted.  (Routine GMTSCNB)
   
   24   Provided flexibility to the users for defining     GMTS*2.7*29 PTM
        the list of Health Summary Types found in the
        Health Summary Reports box of the Reports Tab
        of CPRS GUI.  (GMTSXA* routines, GMTS GUI*
        Options and file #142.98)  NOTE:  This is an 
        enhancement and requires the installation of 
        patch OR*3.0*85 to be fully implemented.
       
   25   Removed references to OERR v2.5 code.          
        (Routines GMTSADH, GMTSADH1, GMTSADHC,
        GMTSADOR, GMTSLROE, GMTSORC, GMTSU, GMTSUP
        and GMTSUP1 - these routines either call
        into routines that no longer exist under 
        OERR 3.0 or access files that no longer 
        exist)

  INSTALLATION:
  =============
  
      This patch can be installed with users on the system, however, it
      is recommended that it be installed at a time of day when a minimum
      number of users will be effected. 
 
      WARNING:  If you respond "YES" to the "Rebuld Menu Trees" prompt in
                step 6, it may take up to two hours as some of the larger
                sites.  During this time BMCA and CPRS are unavailable.
                Until XU*8*156 and XU*8*157 are released, this will remain
                a major problem installing patches with options and menus.
  
      1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu, to
          load the KIDS Transport Distribution global.
  
      2.  Review your mapped set.  If it includes any of the routines 
          listed in the routine summary above, they should be removed 
          from the mapped set at this time.
  
      3.  From the Kernel Installation and Distribution System (KIDS) Menu,
          select the Installation menu.
  
      4.  From the Install menu, you may elect to use the following
          options (when prompted for INSTALL NAME, enter GMTS*2.7*47):
  
            a.  Verify Checksums in Transport Global
            b.  Print Transport Global
            c.  Compare Transport Global to Current System
            d.  Backup a Transport Global
  
      5.  Use the Install Package(s) option and select the package
          GMTS*2.7*47.
      
      6.  When prompted "Want KIDS to Rebuild Menu Trees Upon Completion
          of Install? YES//", respond "YES."  This patch exports new options
          which need to be placed on existing menus.
 
          WARNING:  For some site this may take one or two hours to
                    complete the menu tree rebuild.  During this time
                    BMCA and CPRS are unavailable.  Until XU*8*156
                    and XU*8*157 are released, this is a major problem.
      
      7.  When prompted "Want KIDS to INHIBIT LOGONs during the install?
          YES// ", Respond "NO."
      
      8.  Disabling options and protocols:
      
            When prompted "Want to DISABLE Scheduled Options, Menu
            Options, and Protocols? YES//", respond "YES"
      
              When prompted "Enter options you wish to mark as 
              'Out Of Order':" enter:
      
                     GMTS*
      
              When prompted "Enter protocols you wish to mark as 
              'Out Of Order':" enter:
      
                     GMTS*
      
      9.  If you unmapped any routines as part of step 2, they should be
          returned to the mapped set once the installation has run to 
          completion.
      
     10.  Clean up:  None. Post-Install GMTSP47 is automatically deleted 
          by KIDS upon successful installation of this patch.
