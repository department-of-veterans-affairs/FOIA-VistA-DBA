Subject: CORRECTIONS TO THE PAY ENTITLEMENT (457.5) FILE

Category: 
  - PATCH FOR A PATCH
  - Routine

Description:
============

 This patch addresses the issue reported in NOIS LEX-0104-41744, whereby 
 when patch PRS*4.0*90 was installed, it accidentally created two new
 entries (IENs 2 & 3) in the PAY ENTITLEMENT (#457.5) file, instead of
 updating two of the existing entries (IENs 64 and 106).
  
  
 This patch addresses the following New Service Request(NSR):
 ------------------------------------------------------------
 There are no Service Requests associated with this patch
  
  
 This patch addresses the following NOIS message(s):
 ---------------------------------------------------
 LEX-0104-41744       PRS*4*90 problems
  
  
 Overview of NOIS Message(s):
 ----------------------------
 LEX-0104-41744       PRS*4*90 problems
  
  
 Problem:
 --------
 When patch PRS*4.0*90 was installed, it accidentally created two new 
 entries (IENs 2 & 3) in the PAY ENTITLEMENT (#457.5) file, instead of
 updating two of the existing entries (IENs 64 and 106).
  
  
 Resolution:
 -----------
 This patch will delete these two erroneous entries, and will correctly 
 update the two existing entries.  It will also check the ENTITLEMENT (#4) 
 field in the EMPLOYEE (#459.01) multiple of the TIME & ATTENDANCE (#458) 
 file for PP 04-01 for any entries that point to the two erroneous entries,
 and if any are found, it will point them to the correct entries.

Routine Information:
====================

Routine Name:
  - PRSXP91

 This Pre-Install routine will correct the PAY ENTITLEMENT (#457.5) file. 
 It will then loop through the employees in the TIME & ATTENDANCE 
 (#458) file for PP 04-01 and for the effected employees will update the
 ENTITLEMENT (#4) field of the EMPLOYEE (#458.01) multiple, which is a
 pointer to the PAY ENTITLEMENT (#457.5) file.
