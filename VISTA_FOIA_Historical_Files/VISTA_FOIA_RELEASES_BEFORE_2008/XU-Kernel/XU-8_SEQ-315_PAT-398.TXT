Subject: Clinical Core Registration Screen

Category: 
  - Routine
  - Data Dictionary

Description:
============

 This patch corrects the following issue(s):
  
 1.  The following 4 sites were not being displayed when answering the
 "VHA Training Fac.:" question:
  
  ALASKA VAHSRO           AK      RO-OC   463        
  HONOLULU-RO             HI      RO-OC   480        
  JEFFERSON BARRACKS-RO   MO      RO-OC   587        
  MANILA-RO               PH      RO-OC   358      
  Routine: XUOAAUTL
  Field: VHA TRAINING FACILITY(#12.4)
  
 2.  At the request of VHA Central Office the following question is 
 changed:
   from: Is this person a Registered Trainee?
   to: Is this person an active Trainee?
  Forms: XU-CLINICAL TRAINEE, XUEXISTING USER, XUNEW USER, XUREACT USER
  
 3.  At the request of VHA Central Office the following fields are changed:
  Field: CLINICAL CORE TRAINEE(#12.6)
  DESCRIPTION:
  from: This field designates whether or not the person is a Clinical Core 
        Trainee.
  to: This field designates whether or not the person is an active Trainee.
  'HELP'-PROMPT:
  from: Is this person a Clinical Core Trainee?
  to: Is this person an active Trainee?
  
  Field: DATE NO LONGER TRAINEE(#12.7)
  DESCRIPTION:
  from: This is the date when a Clinical Core Trainee is no longer to be 
        considered a trainee.
  to: This is the date when a Registered Trainee is no longer to be 
      considered a trainee.  This may be at the end of a rotation period or
      at the end of the VA training experience.
  'HELP'-PROMPT:
  from: Answer with the Date the person will no longer be a Clinical Core 
        Trainee.
  to: Answer with the Date when this Trainee became inactive. Future dates 
      are not allowed.

 recommended that it be Queued for a time of generally least activity.
