 Installation Instructions:
 ==========================
  
 1. Users ARE allowed to be on the system during the installation. 
    It will take less than one minute to install the
    patch, but you may queue the installation if you wish.
  
 2. You DO NOT need to stop TaskMan or the background filers.
  
 3. Use the 'INSTALL/CHECK MESSAGE' option on the PackMan menu. This      
    option will load the KIDS package onto your system.
  
 4. The patch has now been loaded into a Transport global on your system. 
    You now need to use KIDS to install the Transport global. On the KIDS
    menu, under the 'Installation' menu, use the following options:
  
                 Verify Checksums in Transport Global
                 Print Transport Global
                 Compare Transport Global to Current System
                 Backup a Transport Global
  
    On the KIDS menu, under the 'Installation' menu, use the following
    option:
                 Install Package(s)  
                 Select INSTALL NAME:XU*8.0*411
                                     ==========
   Want KIDS to INHIBIT LOGONs during the install? YES// NO
                                                         ==
   Want to DISABLE Scheduled Options, Menu Options, and Protocols? YES// NO
                                                                         ==
 5. Post installation:  
    From prompted mode D LOOP^XU8P377D: to list all Users who need to be
    assigned new Person Classes. This will run immediately (no device
    asked). Users may want to turn on "screen capture" for this report.
  
    It will be fine if there is no user listed. If there are some users
    listed, please assigned new Person Classes for the users as soon as
    possible.
