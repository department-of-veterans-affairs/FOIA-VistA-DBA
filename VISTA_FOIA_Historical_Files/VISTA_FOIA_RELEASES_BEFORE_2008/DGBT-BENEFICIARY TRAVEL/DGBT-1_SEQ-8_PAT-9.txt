Subject: DISABLE COREFLS SOFTWARE

Category: 
  - Data Dictionary
  - Routine

Description:
============

 Reversion of CoreFLS software back to VistA software ONLY.
 ==========================================================
  
 This patch will disable access to the CoreFLS software in Beneficiary 
 Travel.  The parameter entry COREFLS ACTIVE will be placed in an 'off' or 
 'no' status and the field will be made uneditable.  The following options 
 will be placed 'out of order' with the message 'CoreFLS Disabled 
 (DGBT*1*9)':
  
 Add a local vendor (COREFLS)     [DGBT LOCAL VENDOR ADD]
 Local Vendor Menu (COREFLS)      [DGBT LOCAL VENDOR MENU]
 Update Local Vendor (COREFLS)    [DGBT LOCAL VENDOR UPDATE]
  
 File: MAS PARAMETERS (#43) Field: COREFLS ACTIVE (#723) will be set to 
 'no' and the field will be made uneditable.
  
 If the option 'Update Local Vendor (COREFLS)' [DGBT LOCAL VENDOR UPDATE] 
 is on your scheduled background task list, it can be removed.
  
  
 This patch addresses the following New Service Request (NSR):
 -------------------------------------------------------------
 There is no NSR associated with this patch.
  
  
 This patch addresses the following NOIS message(s):
 ---------------------------------------------------
 There are no NOIS messages associated with this patch.
