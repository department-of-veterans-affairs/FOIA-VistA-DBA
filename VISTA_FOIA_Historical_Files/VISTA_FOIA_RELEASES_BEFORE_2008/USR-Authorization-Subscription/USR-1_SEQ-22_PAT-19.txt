Subject: ADD NEW USER ROLE (INTERPRETER) TO USR ROLE FILE (#8930.2)

Category: 
  - Data Dictionary
  - Informational

Description: 
============

 This patch is exporting a new user role, INTERPRETER,  required for patch 
 TIU*1*109, TIU ENHANCEMENT FOR CLINICAL PROCEDURES.
  
 The new role will allow creating business rules in ASU to assist with 
 controlling privileges when a Clinical Procedure being performed has been 
 done and is ready for completion. 
  
 The new role is described in more detail in the TIU*1*109 documentation.
  
 Installation:
 =============
 This patch is not being distributed through the National Patch Module. It
 is being distributed in a host file with TIU*1*109.  Please refer to the
 SOFTWARE AND DOCUMENTATION RETRIEVAL and the INSTALLATION INSTRUCTIONS
 sections of the TIU*1*109 patch for the software acquisitions and
 installation instructions.
