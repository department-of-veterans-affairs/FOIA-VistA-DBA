Subject: Code Set FY06 2nd Qtr Update

Category: 
  - Informational

Description:
============

  This patch is an informational patch announcing the release of the 
  CPT/HCPCS coding changes for FY 2006 effective Jan 1, 2006.  This patch
  contains no routines, data, or data dictionaries.  The data for this
  patch is exported with the corresponding Lexicon data in patch 
  LEX*2.0*39.  Patch LEX*2.0*39 requires that patch LEX*2.0*37 be
  installed first.
  
  The data in LEX*2.0*39 affects the following CPT/HCPCS files: 
  
          81         CPT
          81.1       CPT CATEGORY
          81.2       CPT COPYRIGHT
          81.3       CPT MODIFIERS
