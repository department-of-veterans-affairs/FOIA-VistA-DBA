Subject: RECEIVING UNDEFINED ERROR AT LBRY PATRONS REPORTS OPTION.

Description:
============

 ************************** N O T E *********************************
 This patch has *NO* conflict with the CoreFLS environment for Library.
 ********************************************************************
  
 This patch will fix the undefined error created when accessing menu 
 option LBRY PATRONS REPORT for Subject List of Journals Available.

 =================

    Duplicate:

 NOIS OVERVIEW:
 ==============
 1) STL-0598-41795  undef SL+2 ^LBRYRPT1
  
  Problem:
  --------
  
  An undefined error occurs when running the option 
  [LIBRARY PATRON OPTIONS] RFP Reports for Patrons under
  SLJ:Subject list of Journals Available. The TITLE field
  (.01) in the LOCAL SERIALS FILE (#680), does not have a
  corresponding TITLE field (.01) in the TITLE AUTHORITY
  file (#680.5).    
    
  Resolution:
  -----------
  
  Modified routine LBRYPRT1 to access the TITLE field from the 
  LOCAL SERIALS FILE (#680), if the TITLE field from the TITLE
  AUTHORITY file is missing.
