Category: 
  - Routine
  - Enhancement (Mandatory)

Description:
============

 1)  This patch will concatenate the following 3 fields in the RECORD 
 OF PROS APPLIANCE/REPAIR FILE (#660) for the NPPD Roll-Up that may have
 had data added that exceeded the length specified as the maximum extract.
  
 16        REMARKS
 24        DESCRIPTION 
 25        DELIVER TO
  
 No user intervention is required by site personnel
 once the patch is installed at the site.
  
 2) The post install routine ^RMPR126P in this patch will be run to check
 and make sure there is not any corruption to the package file.  This
 routine will be transported with all Prosthetics released patches until
 further notification.
