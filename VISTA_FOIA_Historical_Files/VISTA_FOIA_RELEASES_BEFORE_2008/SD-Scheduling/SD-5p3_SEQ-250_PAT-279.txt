Subject: WAIT LIST TRANSMISSION UPDATES

Category: 
  - Other

Description: 
============

 This patch edits the option SD Wait List Trans to AAC[SD WAIT LIST TRANS 
 TO AAC], making it more compatible with TASKMAN.
  
 Technical Description:
 ---------------------- 
 1. This patch deletes the security key SDWL PARAMETER from option
    SD Wait List Trans to AAC[SD WAIT LIST TRANS TO AAC].
 2. The field ENTRY ACTION (#20), in file Option (#19), is also deleted
    from the SD Wait List Trans to AAC[SD WAIT LIST TRANS TO AAC] option.
 3. The field ROUTINE (#25), is changed to "INIT^SDWLTR" in file
    OPTION(#19) for option SD Wait List Trans to AAC[SD WAIT LIST TRANS TO
    AAC].
  
  
 This patch addresses the following E3R(s):
 ------------------------------------------
 There are no E3Rs associated with this patch
  
  
 This patch addresses the following NOIS message(s):
 ---------------------------------------------------
 1. DUR-0103-30482  EWL Data Transmission to Austin
  
  
 Overview of NOIS Message(s):
 ---------------------------
 1. DUR-0103-30482  EWL Data Transmission to Austin
