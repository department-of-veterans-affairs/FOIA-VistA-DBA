Subject: CORRECT SPANISH TEXT IN MEDICATION INSTRUCTION SHEETS

Category: 
  - Routine

Description:
============

  
 The words information and page in the header of patient medication
 instruction sheets (PMIs) contain incorrect characters when printed in
 Spanish.  This patch corrects that defect.

 Installation Instructions:
 ==========================
  
 Users may remain on the system.  The option Print a PMI Sheet [PSNPMIS
 PRINT] should be placed out of order.  Installation takes less than a
 minute.
  
 1.  Use the INSTALL/CHECK MESSAGE on the PackMan menu.
  
 2.  From the Kernel Installation & Distribution System menu, select
     the Installation menu.  When prompted, select PSN*4.0*141.
  
 3.  From this menu, you may select the options Verify Checksums in
     Transport Global, Backup a Transport Global, and Compare Transport
     Global to Current System.  When prompted, select PSN*4.0*141.
  
 4.  Use the Install Package(s) option and select PSN*4.0*141
  
 5.  When prompted "Want KIDS to INHIBIT LOGONs during the install? YES//"
     respond NO.
  
 6.  When prompted "Want to DISABLE Scheduled Options, Menu Options, and
     Protocols? YES//" respond YES and select the option
  
             Option Name                  Menu Text
             =============                =================
             PSNPMIS PRINT                Print a PMI Sheet
