Subject: Change the Stock On Hand Greater Than range to 270 days

Category: 
  - Other

Description:
============

 This patch updates the 'Stock On Hand Report Greater Than Range' parameter
 for the CLO GIP Reports (CLRS) option.  The CLO office needs to have this
 parameter changed from 90 days to 270 days for reports run starting
 September 1st, 2007.  Installation of this patch will automatically update
 the parameter value.
   
 ASSOCIATED REMEDY TICKETS: 
 =========================
 1.  HD197385  -  Change the Stock On Hand Greater Than range to 270 days
  
 PARTICIPATING TEST SITES:
 =========================
 Bronx, NY
 N. Fla./S. Ga.
 Long Beach, Ca.
     
  
 REMEDY OVERVIEW:
 ================
  
 1.  HD197385  -  Change the Stock On Hand Greater Than range to 270 days
   
 Problem:
 -----------
 The CLO Office wants to utilize the results of the CLO GIP Reports (CLRS)
 option [PRCPLO CLO GIP OPTION] to monitor Engineering related Inventory
 Points in GIP. The 'Stock On Hand Report Greater Than Range' parameter
 needs to be changed from 90 to 270 days.  This could be done manually by
 each site but for the sake of consistency a national patch entry has 
 been requested.
   
 Resolution:
 -----------
 This patch will distribute the new value of 270 for the PRCPLO GREATER
 THAN RANGE entry in the PARAMETERS file [#8989.5]
   
 Items affected:  PRCPLO GREATER THAN RANGE parameter.
