Subject: DISALLOW INVALID STOP CODE VALUES IN FILE #728.44

Category: 
  - Data Dictionary

Description:
============

 The option Enter/Edit DSS Stop Codes for Clinics [ECXSCEDIT] allows
 the DSS Stop Code and the DSS Credit Stop Code fields to be entered
 with no validity check, and it should check that the values entered
 are a valid entry in the CLINIC STOP (#40.7) file.
  
 ASSOCIATED NOIS:
 ================
 1.BED-1003-12069     Invalid Stop code values

 NOIS OVERVIEW:
 ==============
 1. BED-1003-12069     Invalid Stop code values
  
    Problem:
    -------
    The Enter/Edit DSS Stop Codes for Clinics [ECXSCEDIT] option allows
    the DSS Stop Code and the DSS Credit Stop Code fields to be entered
    with no validity check.
  
    Resolution:
    ----------
    The Data Dictionary for CLINICS AND STOP CODES (#728.44) file was 
    modified to verify that new or changed entries for the DSS Stop Code
    and DSS Credit Stop Code fields appear in the CLINIC STOP (#40.7)
    file.
