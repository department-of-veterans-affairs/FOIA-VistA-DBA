Subject: NO RESOLUTION COMMENT ON VA ALERT

Category: 
  - Routine

Description: 
============

 This patch addresses the following NOIS messages:
  
 DAN-0898-41085
 Resolution comments are displaying with the text of the VA Alerts on
 Reports of Contact.  Resolution comments should not print, as they should
 be available only to Patient Advocates.
  
 BYN-0698-10433
 When queuing the Customer Service Standard report totals are not accurate.
  
  
 If installed during the normal work day it is recommended that the
 following menu options (File #19) and all of their descendants be
 disabled to prevent possible conflicts while running the KIDS Install.
 Other DHCP users will not be affected.
  
        [QAC MAIN MENU] - Patient Representative Manager Menu
        [QAC USER] - User Menu
        [QAC ALERT] - Send or Kill an Alert
  
 Install Time - Less than 5 minutes
  
  
          ================INSTALLATION INSTRUCTIONS =================
   
   1.   LOAD TRANSPORT GLOBAL 
        ---------------------
        Choose the PackMan message containing this patch and invoke the 
        INSTALL/CHECK MESSAGE PackMan option.

        step 3 below.

 called, Resolution comments will not display on the VA Alert.
