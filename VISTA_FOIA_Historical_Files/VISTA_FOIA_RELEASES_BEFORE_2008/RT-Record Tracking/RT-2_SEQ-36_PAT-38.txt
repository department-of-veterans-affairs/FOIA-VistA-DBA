Subject: ERROR MESSAGE WHEN BORROWER INFORMATION IS INCOMPLETE

Category: 
  - Routine

Description:
============

 The user gets the following message when trying to select a site
 that is missing the requested printer or remote mail group in
 the borrower setup definition:
  
 ROUTING INFORMATION FOR THIS BORROWER/LOCATION IS INCOMPLETE - 
 SEE SITE MANAGER. NO MESSAGE WILL BE SENT
 **************************************************************************
  
  ASSOCIATED NOIS:
  ================
  
   MAD-0703-40126     Routing Information for Borrower Incomplete
   TOM-1203-40636     Unable to request record
  
  ASSOCIATED E3RS:
  ================
   
  N/A
   
 PARTICIPATING TEST SITES
  ========================
  
 Hampton, VA
 White City, OR
  
  NOIS OVERVIEW:
  ==============
  1. MAD-0703-40126     Routing Information for Borrower Incomplete
     TOM-1203-40636     Unable to request record
  
   Problem: When the BORROWER setup is missing either the
    REQUEST PRINTER or the REMOTE MAIL GROUP the transaction is
    not completed and the user receives a message as noted in the
    description.
  
   Resolution:  Modified the logic at WHOTO^RTTR1 from an 'OR'
    condition to an 'AND' condition.
