Description:
============

   This patch is part of the Code Text Descriptors (CTD) Project. This patch 
   is NOT part of a build and should be installed independently.
   Please review the Installation heading of this patch for Installation 
   Guide retrieval information for the CTD software.
 
   This patch modifies selected data extract routines to take advantage of 
   the Text Versioning APIs exported by ICPT/HCPCS and the DRG Grouper 
   packages.
 
   The following Health Summary routines have been modified to call the
   Code Text Descriptor compliant APIs to display the versioned text:
  
        Routine     Calls              To Get
        =======     ===============    ============= 
        GMTSDCB     $$ICDDX^ICDCODE    Versioned Diagnosis short text
        GMTSDGA     $$ICDDX^ICDCODE    Versioned Diagnosis short text
        GMTSDGC1    $$ICDDX^ICDCODE    Versioned Diagnosis short text
        GMTSDGC2    $$ICDOP^ICDCODE    Versioned Procedure short text
        GMTSDGP     $$ICDOP^ICDCODE    Versioned Procedure short text
        GMTSPXU1    $$ICDDX^ICDCODE    Versioned Diagnosis short text
        GMTSPXU1    $$ICDOP^ICDCODE    Versioned Procedure short text
        GMTSPXU1    $$ICDD^ICDCODE     Versioned Diag/Proc description
        GMTSROE     $$CPT^ICPTCOD      Versioned CPT/HCPCS short text
        GMTSROE     $$MOD^ICPTMOD      Versioned CPT Modifier short text

   Act (HIPAA) Code Text Descriptors (CTD) Installation Guide for details

   independent patches.

   This patch may be installed with users on the system; however, it should
   be installed at a non-peak time to minimize disruption to the users.
   Installation of this patch should take less than 1 minute.
    
   1. Use the INSTALL/CHECK MESSAGE option on the Packman Menu.  
     
   2. Review your mapped set.  If any of the routines listed in this 
      patch are mapped, they should be removed from the mapped set at 
      this time.  
    
   3. From the Kernel Installation and Distribution System (KIDS) Menu, 
      select the Installation menu.  
    
   4. From the Installation menu, you may elect to use the following 
      options (when prompted for INSTALL NAME, enter GMTS*2.7*71): 
    
      a.  Backup a Transport Global 
      b.  Compare Transport Global to Current System 
      c.  Verify Checksums in Transport Global
   
   5. Use the Install Package(s) option and select the package GMTS*2.7*71.
    
   6. When prompted 'Want KIDS to INHIBIT LOGONs during the install? YES//',
      respond NO.
    
   7. When prompted 'Want to DISABLE Scheduled Options, Menu Options, and 
      Protocols?, respond NO.
  
   8. If routines were unmapped as part of step 2, they should be 
      returned to the mapped set once the installation has run to
      completion.
