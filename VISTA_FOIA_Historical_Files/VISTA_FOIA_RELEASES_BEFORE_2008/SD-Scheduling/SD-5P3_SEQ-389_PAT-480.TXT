Subject: Undefined error occurs at INIT+4 SDAMEP1

Category: 
  - Routine

Description:
============

 This patch will fix an undefined variable error that occurs when 
 selecting Expand Encounter within the Correct Incomplete Encounters menu 
 option.
  
  
 ASSOCIATED REMEDY TICKET:
 ========================
 HD132608 - Undefined error occurs at INIT+4^SDAMEP1
  
  
 PARTICIPATING TEST SITES:
 ========================
 W. LA VAMC
 Atlanta VAMC
 Cheyenne VAMC
  
  
 REMEDY TICKET OVERVIEW:
 ======================
 HD132608 - Undefined error occurs at INIT+4^SDAMEP1
  
     Problem:
     --------
     An undefined variable error occurs at INIT+4^SDAMEP1 when the user
     selects Expand Encounter within the Correct Incomplete Encounters
     menu option.
  
     Resolution:
     -----------
     This error occurs because routine SDAMEP, which calls INIT^SDAMEP1,
     is not passing the patient's DFN, as it should.  Routine SDAMEP
     will be modified to pass the patient's DFN to SDAMEP1.

     (when prompted for INSTALL NAME, enter SD*5.3*480):

     b.  Compare Transport Global to Current System - This option will
