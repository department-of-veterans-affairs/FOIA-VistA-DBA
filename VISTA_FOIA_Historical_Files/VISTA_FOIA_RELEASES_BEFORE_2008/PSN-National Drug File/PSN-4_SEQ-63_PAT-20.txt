Subject: CHANGE READ ACCESS ON FILES

Category: 
  - Enhancement (Mandatory)
  - Routine

Description: 
============

  
 This patch changes the read access for four files associated with the
 National Drug File application.
  VA GENERIC file (#50.6)
  VA DISPENSE UNIT file (#50.64)
  NDC/UPN file (#50.67)
  VA PRODUCT file (#50.68)
  
 These files were originally sent with a read access of @.  This patch
 changes the read access to Pp which is similar to that found on other
 pharmacy files.

  Installation Instructions:
  ==========================
  
 This patch can be installed with users on the system.  No options have to be
 placed out of order.  Installation takes less than one minute.  This patch
 can be installed any time.
  
 1.  The routine in this patch is new.  There are no mapping issues.
  
 2.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu.
  
 3.  From the Kernel Installation & Distribution System menu, select
     the Installation menu.  When prompted, select PSN*4.0*20.
  
 4.  From this menu, you may choose to use the option Verify Checksums in
     Transport Global.  When prompted, select PSN*4.0*20.  All components
     of this patch are new, so the option Backup a Transport Global and
     the option Compare Transport Global to Current System do not have to
     be invoked.
  
 5.  Use the Install Package(s) option and select PSN*4.0*20.
  
 6.  When prompted "Want KIDS to INHIBIT LOGONs during the install? YES//"
     respond NO.
  
 7.  When prompted "Want to DISABLE Scheduled Options, Menu Options, and
     Protocols? YES//" respond NO.
