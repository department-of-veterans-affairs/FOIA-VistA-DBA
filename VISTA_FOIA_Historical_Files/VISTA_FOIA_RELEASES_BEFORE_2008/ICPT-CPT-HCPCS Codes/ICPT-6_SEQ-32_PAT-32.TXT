Subject: CPT/HCPCS Code Set FY07 2nd Qtr Update (Informational)

Category: 
  - Informational

Description:
============

  This patch is an informational patch announcing the release of 
  additional CPT/HCPCS Coding changes for FY 2007 2nd quarter, 
  effective Jan 1, 2007. This patch contains no routines, data, 
  or data dictionaries.  The data for this patch is exported with
  the corresponding Lexicon data in patch LEX*2.0*46.  Patch 
  LEX*2.0*46 requires that patch LEX*2.0*45 be previously installed.
     
  The data in LEX*2.0*46 affects the following CPT/HCPCS files: 
     
         CPT/HCPCS Procedure       #81   
         CPT Message               #81.2
         CPT Modifier              #81.3
