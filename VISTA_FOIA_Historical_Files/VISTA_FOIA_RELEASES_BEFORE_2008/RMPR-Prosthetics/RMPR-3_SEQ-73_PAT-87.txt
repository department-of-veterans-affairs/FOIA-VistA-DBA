 The changes and enhancements are listed below:
  
  The purpose of this patch is to create a standalone menu option that will
 allow for the enter/edit of an item's serial number, make and model number
 for tracking and recall needs.
  
  1) Prosthetics IT Task List item: Fix to enter S/N, Make & Model
 (Serial # Menu Option) requires that a menu option be added for users
 to enter the required data.  To satisfy this requirement one
 new menu option, Enter Product Information for Recall [RMPR ENTER
 RECALL DATA], is being added to the Purchasing [RMPR PURCHASING MENU] menu.
  
  2) Prosthetics IT Task List item: Fix to enter S/N, Make & Model
 (Serial # Menu Option) requires that a menu option be used for users
 to enter the required data.  To satisfy this requirement an input
 template, RMPR ENTER RECALL DATA, has been added.  This template is
 attached to the Enter Product Information for Recall [RMPR ENTER
 RECALL DATA] menu option.
  
  3) Prosthetics IT Task List item: Fix to enter S/N, Make & Model
 (Serial # Menu Option) requires that a menu option be used for users
 to enter the required data.  To satisfy this requirement two new fields
 are being added, as described below.  These fields are included in the
 input template RMPR ENTER RECALL DATA, that is used by the menu option.
  Data Dictionary changes:
  
  In file #660  -- RECORD OF PROS APPLIANCE/REPAIR fields #9.1 -- PRODUCT
  DESCRIPTION and #9.2 -- PRODUCT MODEL are being added.
  
   New Fields:
  
  660,9.1       PRODUCT DESCRIPTION    9;1 FREE TEXT
  
               INPUT TRANSFORM:  K:$L(X)>55!($L(X)<1) X
               LAST EDITED:      MAR 02, 2004 
               HELP-PROMPT:      Answer must be 1-55 characters in length. 
               DESCRIPTION:      The manufacture product description to be used
                                 for recalls.  
  
  
  660,9.2       PRODUCT MODEL          9;2 FREE TEXT
  
               INPUT TRANSFORM:  K:$L(X)>55!($L(X)<1) X
               LAST EDITED:      MAR 02, 2004 
               HELP-PROMPT:      Answer must be 1-55 characters in length. 
               DESCRIPTION:      The manufacture product model number to be
                                 used for recalls.

  Installation instruction:
  
  This patch was created with Kernel V8 KIDS and should be installed into
 your production UCI. It can be installed anytime but it is recommended
 that all Prosthetics users should be off the system. Complete 
 installation will take about 5 minutes.
  
  Follow these instructions:                                       
  
  1. Choose the PackMan message containing this patch and invoke the
     INSTALL/CHECK MESSAGE PackMan option.
  
  2. On the KIDS menu, under the 'Installation' menu, use the option
     'Verify Checksum in Transport Global' and verify that all routines
     have the correct checksums. Select Install Name 'RMPR*3.0*87'
  
  3. From the 'Installation' menu of KIDS, use the option 'Install
     Package(s)'.  Select Install Name 'RMPR*3.0*87' and proceed with the
     install.
  
  4. When prompted 'Want KIDS to Rebuild Menu Trees Upon Completion of
     Install? YES//', respond YES.
  
  5. When prompted 'Want KIDS to INHIBIT LOGONs during the install? YES//'
     respond NO.
  
  6. When prompted 'Want to DISABLE Schedule Options, Menu Options and
     Protocols? YES//', respond YES.  When prompted to select the options
     you would like to place out of order, enter the following:
  
                 RMPR OFFICIAL
                 RMPR CLERK
