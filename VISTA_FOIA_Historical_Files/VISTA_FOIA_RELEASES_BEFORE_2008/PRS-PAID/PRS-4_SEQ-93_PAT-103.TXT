Subject: ADD TYPES OF TIME TO LEAVE PATTERN RPT

Category: 
  - Routine

Description:
============

 This patch addresses one issue:
 The Employee Leave Pattern Report does not include all 
 types of leave that are available for a leave request. A
 request was made to include all types of time that are
 available for a leave request on this report.

 REMEDY OVERVIEW
 -------------
 1.    HD89557                 Leave Pattern Report-need addl
                                 types of leave added
       
       Problem:
       -------
       Not all types of time available for a leave request are 
       used in processing the Employee Leave Pattern Report. 
       A request has been made to include all remaining types
       of time available for a leave request in the processing 
       of the Employee Leave Pattern Report.
       
       Resolution
       ----------
       PRSRL4 has been changed to include the following types
       of time:
       AA  Authorized Absence
       AD  Adoption
       CU  Comp Time/Credit Hrs
       DL  Donor Leave
       NL  Non-Pay Annual Leave
       RL  Restored Annual Leave
