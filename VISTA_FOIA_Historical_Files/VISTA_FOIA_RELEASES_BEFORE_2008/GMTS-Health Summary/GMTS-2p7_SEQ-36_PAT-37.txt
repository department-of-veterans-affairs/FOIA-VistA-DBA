Subject: CPT Modifiers and RXOP

Category: 
  - Routine
  - Data Dictionary
  - Enhancement (Mandatory)

Description: 
============

   This patch exports changes to Health Summary Components for PCE, Surgery
   and Radiology.  The principal change is the addition of CPT Modifiers to
   Health Summary components which display CPT Codes and text.
    
    Change                                                Reference  
   -------------------------------------------------------------------------
 
    1)  This build/patch adds CPT Modifiers to six        NOIS MIW-0200-41581
        Health Summary components:           
                                                          SRS CPT Mod Project
           Radiology Profile               RP             Health Summary Sys
           PCE Outpatient Encounters       OE
           Surgery Only Reports            SRO
           Surgery Reports                 SR
           Non-OR Procedures               NSR
           Selected Non-OR Procedures      SNSR
   
              Routines:  GMTSPXOP, GMTSPXU1, GMTSRAE,
                         GMTSRAI, GMTSRAS, GMTSRASP,
                         GMTSRO, GMTSROE, GMTSRON,
                         GMTS1, GMTSADH1, GMTSADH2,
                         GMTSADH4, GMTSADH5, GMTSLOAD,
                         GMTSRM1, GMTSRM5, GMTSU, 
                         GMTSUMX, GMTSUMX2, and GMTSUMX3
 
    2)  It fixes the line feed logic in the component     NOIS RIC-0399-22901
        header for Outpatient Pharmacy (RXOP) when        NOIS LEX-0100-42120
        printing to a host file for GUI applications.     NOIS BAY-0600-30984
                                                          NOIS HUN-0499-22369
                 Routines:  GMTSPSO, GMTSPSO7             NOIS WRJ-0400-12064
 
    3)  It fixes a divide error which occurs at some      NOIS FNC-0500-32040
        sites during the rebuilding of the Adhoc          NOIS MPI-0500-62463
        Health Summary Type during the post-install
        of a new component.
 
              Routines:  GMTSXPD3
 
    4)  Changed direct global reads to read with          DBIA Compliance
        Fileman.
 
              Routines:  GMTSLRPE, GMTSLRME

    INSTALLATION:
    =============
    
      This patch can be installed with users on the system.  Installation
      will take less than three minutes.
      
      1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu, to
          load the KIDS Transport Distribution global.
      
      2.  Review your mapped set.  If any of the routines listed in
          the ROUTINE SUMMARY section are mapped, they should be
          removed from the mapped set at this time.
      
      3.  From the Kernel Installation and Distribution System (KIDS) Menu,
          select the Installation menu.
        
      4.  From the Install menu, you may elect to use the following
          options (when prompted for INSTALL NAME, enter GMTS*2.7*37):
       
          a.  Verify Checksums in Transport Global
          b.  Print Transport Global
          c.  Compare Transport Global to Current System
          d.  Backup a Transport Global
       
      5.  Use the Install Package(s) option and select the package
          GMTS*2.7*37.
    
      6.  When prompted "Want KIDS to INHIBIT LOGONs during the install,"
          respond "No."
 
      7.  When prompted "Want to DISABLE Scheduled Options, Menu Options, 
          and Protocols? YES//", respond "YES."
    
              When prompted "Enter options you wish to mark as 
              'Out of Order':", enter the following:
    
                  GMTS*
    
              When prompted "Enter protocols you wish to mark as
              'Out of Order':", enter the following:
    
                  GMTS*
    
      8.  If routines were unmapped as part of step 2, they should be 
          returned to the mapped set once the installation has run to 
          completion.
    
      9.  The routine GMTSP37 (Post Install Routine) is not automatically
          deleted during the installation and may be deleted at this time.
