Subject: UNDEFINED IN WAIT LIST REPORTING

Category: 
  - Routine

Description:
============

 1.  Undefined error occurs when running the Wait List Report, selecting
     ALL at the institution prompt. then up-arrowing to return to the 
     Institution prompt for editing  
   
 2.  MUMPS error occurs when running Wait List Stat Report and file entry
     has a Wait List Type value of null.
   
 3.  Null subscript error occurs when running the Wait List Report and
     file entry has an Institution or Desired Date of Appointment field
     values of null.

 NOIS OVERVIEW:
 ===============
 ==> SBY-1004-30077  Undefined error running Wait List Report
   
  Problem:
    Undefined error occurs when running the Wait List Report, selecting
    ALL at the institution prompt. then up-arrowing to return to the 
    Institution prompt for editing
       
  Resolution:
    Modify routine SDWLRQ1 to initialize variable SDWLINST
    at the begiining of the query.
   
 ==> BHS-0904-11920  Mumps error running Wait List Stat Report
   
  Problem:
    MUMPS error occurs when running Wait List Stat Report and file entry
    has a Wait List Type value of null.
    
  Resolution:
    Modify routine SDWLRSR, $SELECT command to check
    for a null Wait List Type.   
   
 ==> MAD-1004-40353  Null subscript error running Wait List Report
   
  Problem:
    Null subscript error occurs when running the Wait List Report and
    file entry has an Institution or Desired Date of Appointment field
    values of null.   
 
  Resolution:
    Modify routine SDWLRPT1 to check for null Institution Code
    or Desired Date of Appointment and skip collection of entry.
