Description: 
============

 Patch XM*8*14
  
 NOIS: CMH-0103-N0393
 Test Site: CMOP-Hines
  
 If you run option Historical Queue Data/Stats Report [XMQHIST] after you
 install Kernel patch XU*8*189, the default start date appears as a 5-digit
 number, instead of the current month and year. This patch fixes that.
  
 Note that this patch will work whether patch XU*8*189 is installed or not.
  
 NOTE: This patch may be installed at any time.
 It requires MailMan patch XM*8*8.
 ============================================================================
