Description:
============

    This patch is an informational patch announcing the release of 
    additional CPT/HCPCS Coding changes for FY 2006 4TH quarter, 
    effective Jul 1, 2006. This patch contains no routines, data, 
    or data dictionaries.  The data for this patch is exported with
    the corresponding Lexicon data in patch LEX*2.0*42.  Patch 
    LEX*2.0*42 requires that patch LEX*2.0*44 (Adjust Activation/
    Inactivation Date) CPT Code Sets be installed first.
    
    The data in LEX*2.0*42 affects the following CPT/HCPCS file:
