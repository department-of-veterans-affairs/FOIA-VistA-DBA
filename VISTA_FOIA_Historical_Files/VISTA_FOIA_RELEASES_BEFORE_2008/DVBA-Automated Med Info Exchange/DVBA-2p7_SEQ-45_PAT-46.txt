Subject: UPDATE AMIE WORKSHEETS: 0905, 0910, 0915, 0920, 1440

Category: 
  - Routine

Description: 
============

  This patch updates five AMIE worksheets in the AMIE EXAM file  
  (#396.6). These worksheets provide general guidance to physicians at 
  VA medical centers when performing C&P exams. The implemented changes
  were requested and reviewed by Dr. Caroll McBrine from C&P Regulation
  Office, VACO.
  
  Existing entries in AMIE EXAM file, for the following exams, will be  
  inactivated, and new exams with the same names will be added containing
  the updated information. The revision of four mental disorder worksheets
  consists of a replacement of the former paragraph titled "Competency",  
  with a new paragraph titled "Capacity to manage Financial Affairs".
  Additionally, the spine worksheet was revised.
  
   - MENTAL DISORDERS (except PTSD and Eating Disorders)
   - INITIAL EVALUATION FOR POST-TRAUMATIC STRESS DISORDER (PTSD)
   - EATING DISORDERS (Mental Disorders)
   - REVIEW EXAMINATION FOR POST-TRAUMATIC STRESS DISORDER (PTSD)
   - SPINE(CERVICAL, THORACIC, AND LUMBAR)
  
  Technical:
  ----------
  Existing exams will be inactivated and new exams added to the AMIE  
  EXAM (#396.6) file.  New Reporting Program routines were created using 
  the current routines as a base for the new routines. Additionally the new
  reporting programs for worksheets 0915 and 1440 include modifications
  related to the proper format of printed worksheets. The new routines 
  were modified with the requested updates:
   
  - MENTAL DISORDERS (except PTSD and Eating Disorders) -          WS #0905
     Old Programs: DVBCWME2 calling DVBCWME3
     New Programs: DVBCWME4 calling DVBCWME5
   
  - INITIAL EVALUATION FOR POST-TRAUMATIC STRESS DISORDER (PTSD) - WS #0910 
     Old Programs: DVBCWPD3 calling DVBCWPD4, DVBCWPD5
     New Programs: DVBCWPD6 calling DVBCWPD4, DVBCWPD7
  
  - EATING DISORDERS (Mental Disorders) -                          WS #0915
     Old Programs: DVBCWEA calling DVBCWEA1
     New Programs: DVBCWEA2 calling DVBCWEA3
  
  - REVIEW EXAMINATION FOR POST-TRAUMATIC STRESS DISORDER (PTSD) - WS #0920 
     Old Programs: DVBCWPE3 calling DVBCWPE4, DVBCWPE5
     New Programs: DVBCWPE6 calling DVBCWPE4, DVBCWPE7
  
  - SPINE (CERVICAL, THORACIC & LUMBAR) -                          WS #1440
     Old Programs: DVBCWNS calling DVBCWNS1
     New Programs: DVBCWNS2 calling DVBCWNS3

         ================INSTALLATION INSTRUCTIONS =================
  
 If installed during the normal workday it is recommended that the
 following menu options (File #19) and all of their descendants be disabled
 to prevent possible conflicts while running the KIDS Install. Other VISTA
 users will not be affected.
  
     Regional Office C&P Menu        [DVBA C REGIONAL OFFICE MENU]
     Medical Administration C&P Menu [DVBA C MEDICAL ADMIN C&P MENU]
  
  
     Install Time - less than 5 minutes
  
  
   1.   LOAD TRANSPORT GLOBAL 
        ---------------------
        Choose the PackMan message containing this patch and invoke the 
        INSTALL/CHECK MESSAGE PackMan option.  
              
   2.   DISABLE ROUTINE MAPPING   (DSM for Open VMS sites only) 
        -----------------------
        Disable routine mapping on all systems for the routines listed in 
        step 3 below.  
       
        NOTE:  If the routines included in this patch are not currently in 
        your mapped routine set, please skip this step.

   5.   Select Installation Option: 
        --------------------------
        NOTE:  The following are OPTIONAL - (When prompted for the INSTALL
               NAME, enter DVBA*2.7*46):
       
        a.  Backup a Transport Global - This option will create a backup
            message of any routines exported with this patch. It will not
            backup any other changes such as DD's or templates.
        b.  Compare Transport Global to Current System - This option will
            allow you to view all changes that will be made when this patch
            is installed.  It compares all components of this patch
            (routines, DD's, templates, etc.).
        c.  Verify Checksums in Transport Global - This option will allow
            you to ensure the integrity of the routines that are in the
            transport global.
      
   6.   Select Installation Option: Install Package(s) 
                                    ----------------
        **This is the step to start the installation of this KIDS patch: 
    
        a.  Choose the Install Package(s) option to start the patch
            install.
        b.  When prompted 'Want KIDS to INHIBIT LOGONs during the install?
            YES//' answer NO (unless otherwise indicated)
        c.  When prompted 'Want to DISABLE Scheduled Options, Menu Options,
            and Protocols? YES//' answer YES.
        d.  When prompted 'Enter options you wish to mark as 'Out Of
            Order':'  Enter the following options:
   
            Regional Office C&P Menu        [DVBA C REGIONAL OFFICE MENU]
            Medical Administration C&P Menu [DVBA C MEDICAL ADMIN C&P MENU]
  
        e.  When prompted 'Enter protocols you wish to mark as 'Out Of 
            Order':' press <return>.
  
   7.   REBUILD MAPPED ROUTINE(S)  (DSM for Open VMS sites only)
        =========================
        Optional - Include the routines distributed with this patch
        in the mapped routine set.
      
           NOTE: This step is only necessary if you performed step 2 or 
           if you wish to include the routines in your mapped set.
