Subject: TIU DATA STANDARDIZATION ENHANCEMENTS/FIXES

Category: 
  - Routine
  - Enhancement (Mandatory)
  - Data Dictionary

Description:
============

 This patch introduces several enhancements to the options for mapping 
 local TIU Titles to VHA Enterprise Standard titles, in addition to fixing 
 several problems reported by the field since the release of that Standard.
  
 New functionality is also provided to allow mail bulletins to be sent to
 Clinical Application Coordinators identifying Enterprise Standard Titles
 which have been mapped to Local Titles, and are being inactivated by a
 deployment from ETS.

 Installation Instructions:
 =========================
 This patch may be installed with users on the system; however, it should
 be installed at a non-peak time to minimize disruption to the users.
 Installation of this patch should take less than 2 minutes.
  
 1. Use the INSTALL/CHECK MESSAGE option on the Packman Menu.  
   
 2. From the Kernel Installation and Distribution System (KIDS) Menu, 
    select the Installation menu.  
  
 3. From the Installation menu, you may elect to use the following 
    options (when prompted for INSTALL NAME, enter TIU*1.0*225): 
  
    a.  Backup a Transport Global 
    b.  Compare Transport Global to Current System 
    c.  Verify Checksums in Transport Global
  
 4. Use the Install Package(s) option and select the package TIU*1.0*225.
  
 5. When prompted 'Enter the Coordinator for Mail Group 'TIU CACS': ',
    respond with the name of the primary Clinical Application Coordinator.
  
 6. When prompted 'Want KIDS to INHIBIT LOGONs during the install? YES//',
    respond NO.
  
 7. When prompted 'Want to DISABLE Scheduled Options, Menu Options, and 
    Protocols? YES//', respond YES, and DISABLE the TIU* Options and
    Protocols.
  
 8. When prompted 'Delay Install (Minutes):  (0-60): 0//', respond '0'.
  
  
 Post-Installation Instructions:
 ==============================
 1. Prior to the next weekly deployment of VHA Enterprise Standard Titles,
    have the person identified as the coordinator of the TIU CACS Mail
    Group identify the local users who should belong to the group, and set
    them up as members.
  
 2. After you're satisfied that the patch installed successfully, please
    delete the pre- and post-installation routines, TIUPR225, and TIUPS225.
