Subject: UPDATES FOR 11 AMIE C&P EXAM WORKSHEETS

Category: 
  - Routine

Description:
============

 This patch will put in place and activate 11 new AMIE Compensation and 
 Pension Examination worksheets in the AMIE EXAM file (#396.6).
  
 The worksheets included in this patch are:  Bones (Fractures and Bone 
 Disease) (#1410), Chronic Fatigue Syndrome (#1810), Cranial Nerves 
 (#1205), Cushing's Syndrome (#0415), Eye Examination (#1330), 
 Fibromyalgia (#1445), HIV-Related Illness (#1815), Liver, Gall Bladder, 
 and Pancreas (#0305), Lymphatic Disorders (#0810), Nose, Sinus, Larynx, 
 and Pharynx (#1510), and Rectum and Anus (#0320).

 REMEDY TICKET OVERVIEW:
 =======================
 HD178048 - Updates for 11 AMIE C&P Exam Worksheets
  
    Problem:
    --------
    Dr. Vicki Milton (VBAVACO) has requested that updates be made to 11
    current AMIE Compensation and Pension (C&P) Examination Worksheets:
    Bones (Fractures and Bone Disease) (#1410), Chronic Fatigue Syndrome
    (#1810), Cranial Nerves (#1205), Cushing's Syndrome (#0415), Eye
    Examination (#1330), Fibromyalgia (#1445), HIV-Related Illness (#1815),
    Liver, Gall Bladder, and Pancreas (#0305), Lymphatic Disorders (#0810),
    Nose, Sinus, Larynx, and Pharynx (#1510) and Rectum and Anus (#0320).
  
    Resolution:
    -----------
    a) Created post-install routine DVB121PI, which will deactivate the
       current worksheets and create and activate the new worksheets in
       the AMIE EXAM file (#396.6).  The current and new Internal Entry
       Numbers (IEN) for these worksheets are:
  
       Worksheet                                Current IEN    New IEN
       ---------                                -----------    -------
       1. Bones (Fractures and Bone Disease)    105            212
       2. Chronic Fatigue Syndrome              146            213
       3. Cranial Nerves                        106            214
       4. Cushing's Syndrome                    123            215
       5. Eye Examination                       189            216
       6. Fibromyalgia                          140            217
       7. HIV-Related Illness                   150            218
       8. Liver, Gall Bladder, and Pancreas     195            219
       9. Lymphatic Disorders                   122            220
       10. Nose, Sinus, Larynx, and Pharynx     200            221
       11. Rectum and Anus                      137            222
  
    b) Created routine DVBCWB3 which will contain the new text for the
       AMIE Bones (Fractures and Bone Disease) C&P Exam worksheet.
       Created routine DVBCWB2 which will access the text routine and
       print the worksheet.
  
    c) Created routine DVBCWFS3 which will contain the new text for the
       AMIE Chronic Fatigue Syndrome C&P Exam worksheet.  Created routine
       DVBCWFS2 which will access the text routine and print the work-
       sheet.
  
    d) Created routine DVBCWCN3 which will contain the new text for the
       AMIE Cranial Nerves C&P Exam worksheet.  Created routine DVBCWCN2
       which will access the text routine and print the worksheet.
  
    e) Created routine DVBCWCS3 which will contain the new text for the
       AMIE Cushing's Syndrome C&P Exam worksheet.  Created routine 
       DVBCWCS2 which will access the text routine and print the work-
       sheet.
  
    f) Created routine DVBCWEE4 which will contain the new text for the
       AMIE Eye Examination C&P Exam worksheet.  Created routine DVBCWEE3
       which will access the text routine and print the worksheet.
  
    g) Created routine DVBCWFI3 which will contain the new text for the
       AMIE Fibromyalgia C&P Exam worksheet.  Created routine DVBCWFI2
       which will access the text routine and print the worksheet.
  
    h) Created routine DVBCWHI3 which will contain the new text for the
       AMIE HIV-Related Illness C&P Exam worksheet.  Created routine
       DVBCWHI2 which will access the text routine and print the work-
       sheet.
  
    i) Created routines DVBCWLL6 and DVBCWLL7 which will contain the new
       text for the AMIE Liver, Gall Bladder, and Pancreas C&P Exam work-
       sheet.  Created routine DVBCWLL5 which will access the text routines
       and print the worksheet.
  
    j) Created routine DVBCWLY3 which will contain the new text for the
       AMIE Lymphatic Disorders C&P Exam worksheet.  Created routine
       DVBCWLY2 which will access the text routine and print the worksheet.
  
    k) Created routine DVBCWNW5 which will contain the new text for the
       AMIE Nose, Sinus, Larynx, and Pharynx C&P Exam worksheet.  Created
       routine DVBCWNW4 which will access the text routine and print the
       worksheet.
  
    l) Created routine DVBCWRA3 which will contain the new text for the
       AMIE Rectum and Anus C&P Exam worksheet.  Created routine DVBCWRA2
       which will access the text routine and print the worksheet.

 INSTALLATION INSTRUCTIONS:
 ==========================
 This patch can be installed at any time.  Users do not have to be off the
 system.  Installation will take less than 5 minutes.
  
 1.  Choose the PackMan message containing this patch and invoke the
     INSTALL/CHECK MESSAGE PackMan option.
  
 2.  Start up the Kernel Installation and Distribution System Menu
     [XPD MAIN]:
  
          Edits and Distribution ...
          Utilities ...
          Installation ...
  
     Select Kernel Installation & Distribution System Option:  INStallation
                                                               ---
  
     1    Load a Distribution
     2    Verify Checksums in Transport Global
     3    Print Transport Global
     4    Compare Transport Global to Current System
     5    Backup a Transport Global
     6    Install Package(s)
          Restart Install of Package(s)
          Unload a Distribution
  
 3.  From this menu, you may elect to use the following options
     (when prompted for the INSTALL NAME enter DVBA*2.7*121):
  
     a.  Verify Checksums in Transport Global - This option will allow
         you to ensure the integrity of the routines that are in the
         transport global.
     b.  Print Transport Global - This option will allow you to view
         the components of the KIDS build.
     c.  Compare Transport Global to Current System - This option will
         allow you to view all changes that will be made when this patch
         is installed.  It compares all components of this patch
         (routines, DD's, templates, etc.).
     d.  Backup a Transport Global - This option will create a backup
         message of any routines exported with this patch.  It will not
         backup any other changes such as DD's or templates.
  
 4.  Use the Install Package(s) option and select DVBA*2.7*121.
  
     a.  When prompted "Want KIDS to INHIBIT LOGONs during the install?
         YES//" answer NO.
  
     b.  When prompted "Want to DISABLE Scheduled Options, Menu Options,
         and Protocols? YES//" respond YES.
  
         1) When prompted "Enter options you wish to mark as "Out of
            Order":" enter the following option:
  
            Print Blank C&P Worksheet  [DVBA C PRINT BLANK C&P WORKSHEET]
  
         2) When prompted "Enter protocols you wish to mark as "Out of
            Order":" just press <Return>.
