Subject: Echo Resting BP Range

Category: 
  - Data Dictionary

Description: 
============

 This patch changes the resting blood pressure high/low limits in the the
 ECHO file (#691).
  
  Field # & Name              Old Limits     New Limits
  =====================================================
  3  RESTING SYSTOLIC BP      90 - 220       0 - 300
  4  RESTING DIASTOLIC BP     50 - 150       0 - 300
  
 These fields may be edited via the Enter/Edit Echo (Screen) [MCFSECHO] and
 Line Entry/Edit of ECHO test [MCFLECHO] options.
  
 This change was requested in E3R report number 11833.
  
  Installation Instructions: 
  ==========================
 This patch may be installed with users on the system.  Installation of
 this patch will take less than 5 minutes.
  
  1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu, to
      load the KIDS Transport Distribution global.
  
  2.  From the Kernel Installation and Distribution System (KIDS) Menu,
      select the Installation menu.
  
  3.  From the Install menu, you may elect to use the following
      options (when prompted for INSTALL NAME, enter MC*2.3*20):
  
       a.  Print Transport Global
       b.  Compare Transport Global to Current System
  
  4.  Use the Install Package(s) option and select the package
      MC*2.3*20.
  
  5.  When prompted "Want to DISABLE Scheduled Options,
      Menu Options, and Protocols? YES//", respond "NO"
