Subject: ASU EXPANDING CLASS CAUSES ERROR

Category: 
  - Routine

Description: 
============

 This patch corrects the following problem:
  
 NOIS:  HIN-0201-40669.  When using the menu "User Class Management" [USR
 CLASS MANAGEMENT MENU] and option "User Class Definition" [USER CLASS
 DEFINITION] a problem is encountered when you collapse a user class and it
 is the only class being displayed.  The user would see "undefined variable
 ^TMP("USRCLASS").  This patch corrects this problem.
