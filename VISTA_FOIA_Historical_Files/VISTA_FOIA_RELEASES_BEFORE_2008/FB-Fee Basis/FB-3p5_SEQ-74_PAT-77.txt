  FB-CoreFLS development team.
 *************************************************************************
  
   This patch is part of the Code Text Descriptors (CTD) Project. This
 patch is NOT part of a build and should be installed independently.
  
   The purpose of the patch FB*3.5*77 is to modify the Fee Basis 
 application to assure the validity of text descriptors for each of the
 HIPAA mandated CPT and ICD code sets. Modifications shall be made so that
 the Fee Basis application will use APIs that return active code text
 descriptors for the date of service.
  
 This patch addresses the following New Service Request (NSR):
 -------------------------------------------------------------
 There is no NSR associated with this patch.

 There are no NOIS messages associated with this patch.

 FUNCTIONAL DESCRIPTION
 ======================
 Only CPT (+ HCPCS) code text descriptors, ICD diagnosis (ICD9) code text
 descriptors, ICD procedure (ICD0) code text descriptors and DRG code text
 descriptors that are active for the date of service shall be
 displayed/selectable by the user, regardless of when an entry or edit is
 made. The affected FB options shall be modified to use the new APIs
 including date of service.
  
 TECHNICAL DESCRIPTION
 =====================
 FBAACCB
 Modification has been made to pass the date of service to $CPT^FBAAUTL4 
 call to get correct CPT description.
  
 FBAACO1
 Modification has been made to insure that correct short name will be 
 echoed.
  
 FBAACO1
 Modification has been made to provide date-sensitive short 
 names during 
 look-up.
  
 FBAACP
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBAACR
 Modification has been made to pass the date of service to $CPT^FBAAUTL4 
 call to get correct CPT description.
  
 FBAALU
 Modification has been made to pass the date of service to $$CPTD^ICPTCOD 
 call to get correct CPT description. 
 Modification has been made to pass the date of service to $$MOD^ICPTMOD 
 call to get correct CPT Modifier description.
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBAAMP1
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBAAPCS
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBAAPET
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBAAUTL4
 Modifications have been made to pass the date of service to CPT^FBAAUTL4, 
 MODL^FBAAUTL4, APS^FBAAUTL4 and MOD^FBAAUTL4 to get correct CPT and  CPT 
 Modifier  description.
  
 FBCHEAP
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBCHEP
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBCHPET
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBPHON2
 Modification has been made to pass the date of service to $CPT^FBAAUTL4 
 call to get correct CPT description.
  
 FBPRICE1
 Modification has been made to provide date-sensitive short names during 
 look-up.
  
 FBXIP77E
 Environmental check routine to prevent installation of the patch in  
 CoreFLS sites without coordination with Fee Basis CoreFLS team.
  
 FBCSV1
 Modification has been made to DT2FMDT to allow to convert the date
 strings with leading space.
