Subject: VA-IHD LIPID PROFILE reminder dialog correction

Category: 
  - Informational
  - Other

Description: 
============

 The reminder dialog for the national VA-IHD LIPID PROFILE reminder has a 
 dialog group for lipid profile ordering. 
 
 Health factor ORDER LIPID PROFILE should be added to the VA-LIPID ORDER 
 GROUP dialog group to allow use of the dialog group to be monitored. 
 
 The detailed intructions are as follows:
 
 From the Reminder Manager Menu [PXRM MANAGERS MENU] select menu DM Dialog 
 Management. Within this menu select option DI Reminder Dialogs.
 
 A reminder selection screen is displayed. Select the VA-IHD LIPID PROFILE 
 reminder and on the second screen select the VA-IHD LIPID PROFILE reminder 
 dialog. The reminder dialog will be displayed as shown below. Select the 
 dialog element VA-IHD LIPID ORDER GROUP (item 2) and add the health factor.
