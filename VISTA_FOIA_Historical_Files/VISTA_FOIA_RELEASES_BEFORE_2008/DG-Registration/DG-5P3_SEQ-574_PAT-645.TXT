Subject: TRANSMIT 13 DX CODES FOR PTF

Category: 
  - Routine

Description:
============

 The patch will modify the PTF transmission message to 
 include up to 13 diagnoses from the PTF.
  
  ASSOCIATED NOIS:
  ===============
  HD64124- PRO-0804-10811   PTF CODES
   
  ASSOCIATED E3R(s):
  ================
  N/A
   
  PARTICIPATING TEST SITES:
  ======================

  Hudson Valley HCS
  Upstate NY (Alb,Buff,Syr)
  Providence, RI
  
  NOIS OVERVIEW:
  ==============
  
  1) HD64124- PRO-0804-10811   PTF CODES
  
  Problem:
  --------
  With patch DG*5.3*517, the PTF was modified to collect up to 13
  diagnoses (formerly 10), but the PTF messaging to Austin is only
  sending the first 10.
    
  Resolution:
  -----------
  Modify routine DGPTFTR and DGPTR4 to collect and send up to 13 diagnoses
  in the PTF messaging.

      [XPD MAIN]:

           Installation ...  
     
      Select Kernel Installation & Distribution System Option: INStallation
                                                                
                                                     
           Load a Distribution

           Compare Transport Global to Current System 
           Verify Checksums in Transport Global 
           Install Package(s) 
           Restart Install of Package(s) 
           Unload a Distribution 
           Backup a Transport Global  
   
  3.  From this menu, you may elect to use the following options
      (when prompted for the INSTALL NAME, enter DG*5.3*645):
  
      a.  Backup a Transport Global - This option will create a backup
          message of any routines exported with this patch. It will not
          backup any other changes such as DD's or templates.
      b.  Compare Transport Global to Current System - This option will
          allow you to view all changes that will be made when this patch
          is installed.  It compares all components of this patch
          (routines, DD's, templates, etc.).
      c.  Print Transport Global - this option will allow you to view the
          components of the KIDS build.

          you to ensure the integrity of the routines that are in the
