Subject: HL7 Update

Category:

Description: 
============

 This patch contains the following updates:
  
 1. Routine KMPDHU02 now calls $$CM2^HLUCM().  This is a new api provided
    by the HL7 developer.
 2. CM Tools now gathers asynchronous, as well as sychronous, HL7 data.
 3. Option [KMPD STATUS] is now titled  'Check CM Tools Environment' and 
    has an expanded display (example provided below).
 4. New fields have been added to file #8973.1 (CM HL7 DATA) for hourly 
    data.

 Installation instructions follow:
  
                                 *** NOTE ***
                       All sites must install this patch.
  
 1. This patch can be installed with users on the system.  However, it
    should NOT be installed while the CM Tools Background Driver [KMPD
    BACKGROUND DRIVER] option is running (this job is queued to run every
    day, usually at 1:30 a.m.).
  
 2. Use KIDS to install this patch.
  
      On the KIDS menu, under the 'Installation' menu, use the following
      options:
  
           Verify Checksums in Transport Global
           Print Transport Global
           Compare Transport Global to Current System
           Backup a Transport Global
  
 3. Users can remain on the system.
  
 4. On the KIDS menu, under the 'Installation' menu, use the following
    option:
  
          Install Package(s)  'KMPD*1.0*1'
                               ==========
  
          No Options or Protocols need to be placed out-of-order.
  
          Want to DISABLE Scheduled Options, Menu Options, and Protocols?
          NO
