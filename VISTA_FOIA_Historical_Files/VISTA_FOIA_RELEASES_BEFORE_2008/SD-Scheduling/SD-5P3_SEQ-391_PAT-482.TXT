Subject: SDWL Reports not screening on Institution/Team

Category: 
  - Routine

Description:
============

 When the PCMM Team/Position Wait List (Sch/PCMM) Report is selected, the
 user is first asked to Select an Institution.  If the report category is
 Team, the user is then prompted for Teams.  All Teams, not just those
 specific to the Institution selected, are displayed and can be selected.
 Also, inactive Teams are displayed and can be selected.  This patch will 
 make it so that only the active Teams for the selected Institution are
 displayed.

 REMEDY TICKET OVERVIEW:
 ======================
 HD138788
  
 Problem: 
 -------
 When the PCMM Team/Position Wait List (Sch/PCMM) Report is selected, the
 user is first asked to Select an Institution.  If the report category is
 Team, the user is then prompted for teams.  All teams (not just those
 specific to the Institution selected) are displayed and can be selected.
  
 Resolution:
 ----------
 Modify routine SDWLRQ2 to correctly display only those teams that are 
 in an active status and set up with the Institution selected.
