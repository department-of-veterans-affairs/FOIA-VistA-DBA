 PROJECT OVERVIEW
 ================
 The HVE Phase II project consists of the following functional elements:
  
   1. Remove BEC as a Selectable Branch of Service
   2. Validate WWII Period for Merchant Seaman BOS
   3. MSD Field-Specific Requirements 
   4. Modify Registration Consistency Checks for MSE Data
   5. Modify VistA Military Service Date Field Relationship Requirements
   6. Modify Consistency Checks for Military Service Date Field 
      Relationships
   7. Continuous Enrollment Rules Modifications
   8. Filipino Vet (FV) Data
   9. Add Consistency Checks for Filipino Vets and Merchant Seaman BOS
  10. Rated Disabilities Process Changes
  11. Modify Catastrophic Disabilities Evaluation Process
  12. Modifications to Agent Orange Location
  13. Date of Birth for Mexican Border War and WWI Eligibility
  14. Date of Death Acceptance
  15. Inconsistent Data Elements
  16. Enrollment Group Threshold History
  17. Pending Adjudication MT Expire When GMT Copay
  18. Combat Veteran End Date sharing with HEC
  
 For a more detailed explanation of the actual functionality, please refer
 to the DG*5.3*451 patch description.
  
 FUNCTIONAL DESCRIPTION
 ======================
 The functional changes that will be introduced with this patch include:
  
 1. Modifications were made to allow the expiration of a means test that 
 is pending adjudication in a location where the GMT threshold is greater 
 than the MT threshold.  Means test renewal letters will now be sent to 
 these patients.
  
 2. Modifications were made to the Military Service Data, Screen 1, of the 
 LTC Copayment Test to display the Filipino Vet Proof field if the patient 
 has a Filipino Veteran branch of service.
