 This patch is being released to enhance the insurance API (released in 
 IB*2*228) in response to additional data elements requested by various
 nationally released software packages.
  
 The following parameters are being added:
  
     Status Filter Value:         Description: 
             B                    Indemnity plans included. Default is 
                                  Non-Indemnity plans only.  
  
  
 Data Fields: 
 ID  FIELD NAME              DATA TYPE      FORMAT      DESCRIPTION
 --  --------------------    ------------   ---------   -----------
 18  GROUP NUMBER            TEXT           Free Text   Group number of policy.
  
 19  PATIENT REL TO SUBSC.   NUMBER^TEXT    ID^name     The relationship of the
                                                        patient to the 
                                                        subscriber.  
  
 20  VA ADVANTAGE PLAN/      NUMBER^NUMBER  ID^ID       Until further 
     TRICARE PLAN                                       clarification, the API
                                                        will return a zero (0)
                                                        only for VA Advantage.
                                                        If the plan is TRICARE
                                                        according to the TYPE
                                                        OF PLAN field, a one
                                                        (1) is returned; 
                                                        otherwise zero (0) in
                                                        the second piece.  
  
  
  
  
  
 Technical Description 
 --------------------
 IBBAPI is being updated to enhance the encapsulation insurance API. The 
 new routines IBBDOC and IBBFAPI are also included for the insurance API.
