   ********************* IMPORTANT NOTE *************************** 
   
   This patch ONLY applies to DSS sites that have installed DSS 
   EXTRACTS V3.0 (dated DEC 22,1997 on the second line of routines).  
    
   ****************************************************************
    
    
   The purpose of this enhancement patch is to update the NATIONAL
   CLINIC file (#728.441) with new clinic codes as directed by the
   DSS Program Office.
   
   A total of 158 new records will be added to file #728.441 during
   the post-installation routine included in this patch.  A list of
   the new codes is included below.
   
   Please note that your site's file #728.441 should have exactly
   111 records PRIOR to installation of this patch, and the highest
   record number should be 112.  If this is not the case, you will 
   be unable to install ECX*3*7.  An error message will be displayed
   as follows:
   
        It appears that you may have a problem with File #728.441.
   
        Perform a FileMan print of the entire file which shows
        the internal entry number and #.01 field of each record.
   
        Call Customer Support for further assistance.
        Have the FileMan printout at hand when you call.
   
   Upon completion of ECX*3*7 installation, your site's file #728.441
   should have exactly 269 records, and the highest record number must
   be 270.  If this is not the case, please contact Customer Support.

   INSTALLATION INSTRUCTIONS:
   ==========================
   
   This patch can be loaded with users on the system. 
   
   Installation will take less than 5 minutes.  
    
   1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu to unload
       the KIDS distribution included with this message.
      
   2.  From the Kernel Installation and Distribution System Menu, select 
       the Installation menu.  
    
   3.  From this menu, you may elect to use the following options
       (when prompted for INSTALL NAME, enter ECX*3.0*7):
   
       a. Compare Transport Global to Current System - This option will 
          allow you to view all changes that will be made when this patch 
          is installed. It compares all components of this patch 
          (routines, DD's, templates, etc.).  
       b. Verify Checksums in Transport Global - This option will allow 
          you to ensure the integrity of the routines that are in the 
          transport global.  
    
   4.  Use the Install Package(s) option and select the package ECX*3.0*7.
   
   5.  When prompted 'Want to DISABLE Scheduled Options, Menu Options, and
       Protocols? YES//', respond YES.  When prompted to select the options
       you would like to place out of order, enter the following:
      
       ECX NATIONAL CLINIC          National Clinic Codes for DSS List
      
   6.  If routines were unmapped as part of step 2, they should be returned
       to the mapped set once the installation has run to completion.
  
   7.  After the patch is successfully installed, you may delete routines
       ECX3P7P1 and ECX3P7PT from your system.
