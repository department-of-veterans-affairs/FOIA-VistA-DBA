Subject: UNDEFINED ERROR DURING MAPPING REPORT

Category: 
  - Routine

Description:
============

 When printing a mapping report in Map Pharmacy Locations [PSU MAP PHARMACY
 LOCATIONS] option, an undefined error occurs.
  
  
 Associated Remedy Ticket: 
 =========================
 HD00000000124756
  
 Associated Test Sites:
 ======================

 Remedy Ticket Overview:
 =======================
 Problem:
 When using Map Pharmacy Locations [PSU MAP PHARMACY LOCATIONS] option, an
 undefined error occurs. The user must choose a printer device name that
 does not start with a 'C'. In order to replicate this problem, the user
 must enter the following at the 'DEVICE:' question: DEVICE NAME;80;5
  
 Solution:
 The undefined error will not occur after the patch is loaded.

            (routines, DD's, templates, etc.).

            message of any routines exported with this patch. It will not
