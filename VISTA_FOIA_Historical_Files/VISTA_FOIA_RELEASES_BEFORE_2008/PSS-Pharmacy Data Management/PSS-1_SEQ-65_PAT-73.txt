Subject: STOP PRINTING COMMON BRAND NAMES ON PMI SHEETS

Category: 
  - Routine

Description: 
============

  Several sites have noted that the common brand names printed on a
  Patient Medication Information (PMI) Sheet are not always appropriate.
  This patch stops the printing of common brand names on PMI Sheets printed
  in conjunction with the laser labels.
  
  ****NOTE**** The stand alone option Print a PMI Sheet [PSNPMIS PRINT]
  is not modified by this patch.

  Installation Instructions:
  ==========================
  
  Users may be on the system when this patch is installed. Installation takes
  less than one minute.  This patch can be installed during normal operating
  hours.
  
  
  
   1. Use the INSTALL/CHECK MESSAGE option in the PackMan function.
  
   2. All components of this patch are new so there are no mapping issues.
  
   3. From the Kernel Installation & Distribution System Option, select
     the Installation Option.
  
   4. From this option, you may select to use the option Verify
     Checksums in Transport Global.  All components in this patch are new
     so the options Backup a Transport Global and Compare Transport Global
     to Current System do not have to be invoked.
  
   5. Use the Install Package(s) option and select the package PSS*1.0*73.
  
   6. When prompted "Want KIDS to INHIBIT LOGONs during the install? YES//"
     respond NO.
   
   7. When prompted "Want to DISABLE Scheduled Options, Menu Options, and
     Protocols? YES//" respond  NO.
