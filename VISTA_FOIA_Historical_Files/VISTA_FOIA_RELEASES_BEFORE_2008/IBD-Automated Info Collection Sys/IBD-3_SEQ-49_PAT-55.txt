Subject: CHANGE ARRAY TO TMP GLOBAL

Category: 
  - Routine
  - PATCH FOR A PATCH

Description:
============

 Patch IBD*3.0*51 Code Set Versioning for AICS added routine IBDF18A2,
 which uses a local array for storage.  Some encounter forms are extremely
 large and can generate an allocation error.  This patch will no longer use
 the ARY1 array but use ^TMP("IBDCSV",$J instead.

 There is no NSR associated with this patch.
