Subject: UNDEFINED ERROR WHEN TRYING TO EDIT DISCHARGE SUMMARY

Category: 
  - Input Template

Description:
============

 Any option using the input template TIU ENTER/EDIT DS has the potential 
 to encounter an undefined error if the field Patient Movement Record is 
 null.
  
 ASSOCIATED TICKET:
 ==================
 HD 105470
 HD 106824
  
 PARTICIPATING TEST SITES:
 =========================
 Central Plains HCS
 Huntington,WV
 Spokane,WA
  
 REMEDY OVERVIEW:
 ==============
 Undefined errors occurred when trying to edit a discharge summary, they 
 also occurred when adding an amendment to the discharge summary.  These 
 errors were caused by field Patient Movement Record (1401) in the TIU 
 Document File (8925) being null.
  
 Problem:
 ========
 Undefined errors when using input template TIU ENTER/EDIT DS.
  
 Resolution:
 ===========
 Change the reference in the input template for Patient Movement Record to 
 use $G on variable TIU("TS").
