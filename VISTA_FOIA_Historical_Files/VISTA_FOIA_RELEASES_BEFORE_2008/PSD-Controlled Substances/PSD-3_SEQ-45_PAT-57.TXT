Subject: NURSE SHIFT LOG DOES NOT MATCH INSPECTORS LOG

Category: 
  - Routine

Description:
============

 This patch will correct the problem of the Nursing Shift Check Log [PSD NURSE
 SHIFT LOG] not being consistent with the Inspector's Log for Controlled
 Substances Inspectors Log [PSD PRINT INSPECTOR LOG].
  
 ASSOCIATED NOIS:
 ================
 PTH-0205-20398 Transaction not showing up = Remedy Ticket HD68773
  
 ASSOCIATED E3RS:
 ================
 N/A
  
 PARTICIPATING TEST SITES
 ========================
 CENTRAL TEXAS HCS
 PITTSBURGH HCS
 VA GREATER LOS ANGELES HCS
  
 NOIS OVERVIEW:
 ==============
 PTH-0205-20398 Transaction not showing up = Remedy Ticket HD68773
  
     Problem:
     --------
     Site is having a problem with a transaction 
     not appearing on a Nursing Shift Check Log. 
     The transaction  is there, but is not showing up.
  
     Resolution:
     -----------
     Modify the Nursing Shift Check Log to include 
     missing transactions.
  
  Technical:
  ==========
  PTH-0205-20398 Transaction not showing up
  
     Problem:
     --------
     The Nursing Shift Check Log is inconsistant with the 
     Inspector's Log.  It does not include transactions
     where the Green sheet was not printed or has a status
     of DELIVERED - PERPETUAL INVENTORY.
  
     Before;
     -------
     LOOP+5^PSDNSCL 
        .Q:'$P($G(^PSD(58.81,+$O(^PSD(58.81,"D",+PSDPN,0)),"CS")),U,4)
  
     After:
     ------
     START+4^PSDNSCL   * ADD *
     S STAT=13 F PSD=0:0 S PSD=$O(^PSD(58.8,"AC",STAT,PSD)) Q:'PSD  D LOOP
  
     LOOP+5^PSDNSCL    * DEL *
    .Q:'$P($G(^PSD(58.81,+$O(^PSD(58.81,"D",+PSDPN,0)),"CS")),U,4)
