  This patch contains the following enhancements and problem resolutions:
  
      1)  New option prints HBPC Patient Days of Care by Date Range Report
          [HBHCRP23], located on Reports Menu [HBHC REPORTS MENU] option.
  
          Solution/Routine(s) changed:  HBHCRP23
  
  
      2)  New option prints HBPC Expanded Program Census Report [HBHCRP24],
          located on Census Reports Menu [HBHC CENSUS REPORTS MENU] option,
          which is located on Reports Menu [HBHC REPORTS MENU] option.
  
          Solution/Routine(s) changed:  HBHCRP24
  
  
      3)  New option prints HBPC Address Included Program Census Report
          [HBHCRP25], located on Census Reports Menu [HBHC CENSUS REPORTS
          MENU] option, which is located on Reports Menu [HBHC REPORTS MENU]
          option.
  
          Solution/Routine(s) changed:  HBHCRP25
  
  
      4)  Added alphabetical by patient sort choice to Admissions/Discharges
          by Date Range Report [HBHCRP7] option, located on Reports Menu
          [HBHC REPORTS MENU] option.
  
          Solution/Routine(s) changed:  HBHCRP7
  
  
      5)  In Episode of Care/Length of Stay Report [HBHCRP12] option,
          located on Reports Menu [HBHC REPORTS MENU] option, resolved issue
          of episodes with Discharge Date greater than report ending date
          being omitted from report; fixed date calculation to be ending
          date inclusive for active episodes; changed display sequence of
          totals to match HBPC Patient Days of Care by Date Range Report
          [HBHCRP23] option; fixed a couple of formatting problems. 
  
          Solution/Routine(s) changed:  HBHCRP12
  
  
      6)  Resolved date miscalculation which omitted Jan 31 from date range
          in HBH Mail Message following Build without Errors utilizing Build
          /Verify Transmission File [HBHCFILE] option, located on
          Transmission Menu [HBHC TRANSMISSION MENU] option.

      Patch Installation:
      ==================
  
      Only HBPC  (Home Based Primary Care, formerly HBHC Hospital Based
      Home Care)  users need to be off the system.  Patch installation time
      will be minimal.  There is no recommended time of day for
      installation.
  
        1.  If the above routines are mapped, disable mapping for them.
  
        2.  Use INSTALL/CHECK MESSAGE option on the PackMan menu to load the
            KIDS patch onto the system.
  
        3.  From Kernel Installation and Distribution System Menu, select
            the Installation menu.
  
            From Installation menu, you may choose to use the following
            options prior to installation, select INSTALL NAME  HBH*1.0*21
  
              a.  Verify Checksums in Transport Global
  
              b.  Compare Transport Global to Current System
  
              c.  Backup a Transport Global.
  
        4.  Use Install Package(s) option under the Installation menu,
            select package  HBH*1.0*21.
  
            Answer Yes to the prompt Want KIDS to Rebuild Menu Trees Upon
            Completion of Install.
  
            Answer NO to the prompt Want KIDS to INHIBIT LOGONs during the
            install.
  
            Answer NO to the prompt Want to DISABLE Scheduled Options, Menu
            Options, and Protocols.
  
        5.  On a mapped system, rebuild your map set.
