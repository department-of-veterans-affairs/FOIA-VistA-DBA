Subject: <UNDEFINED> IN PXUTLSTP WHEN DELETING PCE LINE ITEMS

Category: 
  - Routine

Description:
============

 This will correct <UNDEFINED> errors in routine PXUTLSTP when
 deleting line items in PCE Encounter item deletion action 'DE'.

 REMEDY OVERVIEW:
 ================
 HD0000000064573  <UNDEFINED> errors in routine PXUTLSTP
  
  
 Problem:
 -------
   
 When using the PXCE ENCOUNTER ENTRY & DELETE option, action 
 DE - Delete Item,  the delete process will error when deleting the 
 Stop/Credit Stop item in routine PXUTLSTP.
  
   
 Resolution:
 ----------
  
 Modify routine PXUTLSTP by adding a $GET check when examining
 the VISIT File (#9000010) for Credit Stop check to prevent an undefined
 error if 150 node already deleted.
