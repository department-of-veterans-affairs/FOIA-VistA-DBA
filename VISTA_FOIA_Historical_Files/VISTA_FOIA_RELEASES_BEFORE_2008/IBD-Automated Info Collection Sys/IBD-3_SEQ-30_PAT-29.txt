Subject: MISCELLANEOUS FIXES

Category: 
  - Routine
  - Data Dictionary

Description: 
============

 This patch resolves the following 8 NOIS issues for AICS V3.0.
  
 1.  It was reported that while printing encounter forms the home device
     was no longer allowed. 
  
     NOIS Reference: HOU-0499-72617 
  
 2.  During AICS Manual Data Entry options, it was reported if a '??' was
     entered at the checkout date prompt, the option would error with an
     undefined error.
  
     NOIS References: DAY-0799-42032 and LON-1298-61905
  
 3.  When using IBDF SCANNED W/BILL GEN option, a date range is not asked.
  
     NOIS Reference: SAM-0498-20979
  
 4.  When a Means Test Status for a patient is REQUIRED, the 1010F does not
     print. 
  
     NOIS References: CMO-1297-41424 and PRE-0699-62579
  
 5.  On occasion a Clinic Name is defaulted at the "EDIT FORMS FOR WHICH
     CLINIC?" prompt while using IBDF CLINIC SETUP/EDIT FORMS OPTION.  
  
     NOIS Reference: IOW-0198-42459
  
 6.  Neither the option help text nor current documentation states that
     file 359.3, AICS ERROR AND WARNING LOG, is purged via options IBDF MANUAL
     PURGE FORM TRACK or IBDF AUTO PURGE FORM TRACKING.  VISTA on-line
     documentation (http://vista.med.va.gov/fms/aics User Manual) has been
     enhanced to reflect the purging of file 359.3.
  
     NOIS Reference: POR-0399-52001
